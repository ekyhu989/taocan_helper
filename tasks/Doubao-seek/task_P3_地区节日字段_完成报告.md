## 技术实现细节

### 节日标签映射
```javascript
const getFestivalLabel = (festival) => {
  const labels = {
    spring: '春节',
    eid: '古尔邦节',
    nowruz: '肉孜节',
    other: '节日',
  };
  return labels[festival] || '节日';
};
```
