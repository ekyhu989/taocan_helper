## 📦 项目信息

- **版本**: V1.6
- **项目路径**: `e:/taocang_helper`
- **ZIP包**: `e:/taocang_helper_v1.6.zip` (55MB)

## 🚀 快速开始

### 方式1：直接使用现有项目（推荐）

项目已在 `e:/taocang_helper` 目录下配置完成，直接运行：

```bash
cd e:/taocang_helper
npm run dev
```

访问：http://localhost:3000

### 方式2：从ZIP解压使用

1. 解压 `taocang_helper_v1.6.zip` 到任意目录
2. 进入项目目录：
   ```bash
   cd taocang_helper
   ```
3. 安装依赖（如未安装）：
   ```bash
   npm install
   ```
4. 启动开发服务器：
   ```bash
   npm run dev
   ```
5. 访问：http://localhost:3000
