## 📋 文档概述

本设计规范定义了淘仓助手V3.0的完整视觉语言和组件系统，目标是从"AI玩具风格"升级为"政企专业工具风格"。

---

## 🎨 一、设计理念

### 1.1 核心原则

| 原则 | 说明 | 反例 |
|------|------|------|
| **专业克制** | 减少装饰性元素，突出内容本身 | ❌ 渐变背景、大圆角、重阴影 |
| **信息优先** | 内容>容器，功能>形式 | ❌ 过度包装的卡片 |
| **一致性** | 统一设计语言，避免碎片化 | ❌ 不同页面风格不一 |
| **可访问性** | 支持键盘导航、屏幕阅读器 | ❌ 仅支持鼠标操作 |

### 1.2 设计关键词

```
专业 · 克制 · 清晰 · 一致 · 高效
```

### 1.3 设计目标

- ✅ 用户感知：从"AI生成的工具"到"专业政企软件"
- ✅ 视觉评分：从5/10提升到8/10
- ✅ 操作效率：减少30%的视觉干扰

- 主色（藏青）：不超过页面面积的15%

```
信息：操作提示、帮助说明、新手引导
```

**3. 色彩对比度要求**（WCAG AA标准）：
- 正文文字：对比度≥4.5:1
- 大号文字（≥18px）：对比度≥3:1
- UI组件和图形元素：对比度≥3:1

### 2.3 间距系统

#### 基础单位：4px

| Token | 值 | 用途 | 示例 |
|-------|-----|------|------|
| `--spacing-xs` | 4px | 图标与文字间距、紧凑列表 | [4px] |
| `--spacing-sm` | 8px | 表单字段间距、按钮内边距 | [8px] |
| `--spacing-md` | 16px | 卡片内边距、段落间距 | [16px] |
| `--spacing-lg` | 24px | 区块间距、模态框内边距 | [24px] |
| `--spacing-xl` | 32px | 页面区块间距 | [32px] |
| `--spacing-2xl` | 48px | 大区块间距 | [48px] |
| `--spacing-3xl` | 64px | 页面顶部间距 | [64px] |

#### 间距使用规则

**1. 统一使用Token**：
```css
/* ✅ 正确 */
padding: var(--spacing-md);
margin-bottom: var(--spacing-lg);

/* ❌ 错误 */
padding: 15px;
margin-bottom: 17px;
```

**2. 间距优先级**：
- 组件内部：使用xs/sm
- 组件之间：使用md/lg
- 页面区块：使用xl/2xl/3xl

**3. 垂直节奏**：
- 所有垂直间距优先使用偶数倍（8px、16px、24px）
- 保持视觉节奏一致

### 2.4 圆角系统

| Token | 值 | 用途 | 示例 |
|-------|-----|------|------|
| `--radius-sm` | 2px | 按钮、标签、徽章 | [■■] |
| `--radius-md` | 4px | 卡片、输入框、下拉菜单 | [■■■] |
| `--radius-lg` | 8px | 模态框、弹窗、大容器 | [■■■■] |
| `--radius-full` | 50% | 头像、圆形按钮、徽标圆点 | [●] |

#### 圆角使用规则

**1. 一致性原则**：
- 同一层级的组件使用相同圆角
- 避免混用不同圆角

**2. 场景匹配**：
```
小元素（按钮、标签）→ radius-sm (2px)
中元素（卡片、输入框）→ radius-md (4px)
大元素（模态框、容器）→ radius-lg (8px)
圆形元素 → radius-full (50%)
```

**3. 专业感**：
- V3.0减少圆角使用，体现专业感
- 避免V2.0的rounded-2xl（16px）过大圆角

### 2.5 阴影系统

| Token | 值 | 用途 | 示例 |
|-------|-----|------|------|
| `--shadow-none` | `none` | 默认状态（无阴影） | - |
| `--shadow-sm` | `0 1px 3px rgba(0,0,0,0.12)` | 卡片hover态、轻微浮起 | 轻 |
| `--shadow-md` | `0 4px 6px rgba(0,0,0,0.1)` | 下拉菜单、工具提示 | 中 |
| `--shadow-lg` | `0 10px 20px rgba(0,0,0,0.15)` | 模态框、弹窗 | 重 |

#### 阴影使用规则

**1. 默认无阴影**：
- V3.0默认不使用阴影
- 仅在需要表达层级时使用

**2. 使用场景**：
```
卡片：默认无阴影，hover时添加shadow-sm
下拉菜单：默认shadow-md
模态框：默认shadow-lg
浮动按钮：默认shadow-sm
```

**3. 避免过度**：
- ❌ 不要同时使用边框+阴影
- ❌ 不要多层阴影叠加
- ✅ 优先使用边框表达边界

### 2.6 动画系统

#### 过渡时间

| Token | 值 | 用途 |
|-------|-----|------|
| `--transition-fast` | 150ms | 小交互（hover、焦点） |
| `--transition-base` | 250ms | 中等交互（展开、收起） |
| `--transition-slow` | 350ms | 大交互（页面切换、模态框） |

#### 缓动函数

| Token | 值 | 用途 |
|-------|-----|------|
| `--easing-ease` | `ease` | 一般过渡 |
| `--easing-ease-in-out` | `ease-in-out` | 进入/退出动画 |
| `--easing-ease-out` | `ease-out` | 进入动画 |
| `--easing-ease-in` | `ease-in` | 退出动画 |

#### 动画使用规则

**1. 性能优先**：
- 仅使用`transform`和`opacity`做动画（GPU加速）
- 避免使用`width`、`height`、`top`、`left`（触发重排）

**2. 尊重用户偏好**：
```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

**3. 有意义的动画**：
- ✅ 状态变化（加载、成功、错误）
- ✅ 空间关系（展开、收起、滑动）
- ❌ 纯装饰性动画（闪烁、跳动）

## 🧩 三、组件规范

### 3.1 按钮（Button）

#### 按钮类型

| 类型 | 用途 | 样式 | 示例 |
|------|------|------|------|
| **Primary** | 主要操作（每页仅1个） | 藏青背景+白字 | [提交方案] |
| **Secondary** | 次要操作 | 白底+藏青边框+藏青字 | [保存草稿] |
| **Ghost** | 弱操作 | 透明背景+藏青字 | [取消] |
| **Danger** | 危险操作 | 白底+红边框+红字 | [删除] |

#### 按钮尺寸

| 尺寸 | 高度 | 内边距 | 字号 | 用途 |
|------|------|--------|------|------|
| **Large** | 48px | 0 24px | 16px | 主要CTA |
| **Medium** | 40px | 0 20px | 14px | 默认按钮 |
| **Small** | 32px | 0 16px | 13px | 表格操作、紧凑布局 |

#### 按钮状态

| 状态 | 背景色 | 文字色 | 边框 | 说明 |
|------|--------|--------|------|------|
| **Default** | `#1E3A5F` | `#FFFFFF` | none | 默认态 |
| **Hover** | `#152945` | `#FFFFFF` | none | 悬停态 |
| **Active** | `#0F1D30` | `#FFFFFF` | none | 按下态 |
| **Disabled** | `#BDBDBD` | `#FFFFFF` | none | 禁用态 |
| **Loading** | `#1E3A5F` | `transparent` | none | 加载中（显示Spinner） |

#### 按钮规范

**1. 文字规范**：
- 按钮文字使用动词+名词（如"提交方案"、"导出PDF"）
- 避免模糊文字（如"确定"、"提交"）
- 文字长度：2-6个汉字

**2. 数量规范**：
- 每组按钮：最多3个
- 每页Primary按钮：仅1个
- 超过3个使用下拉菜单

**3. 间距规范**：
- 按钮组间距：8px（水平）、12px（垂直）
- 按钮与表单间距：24px
