# V2.0 用户使用手册

## 📋 手册概述

- **产品名称**: AI采购方案生成工具 V2.0
- **版本**: 2.0.0
- **最后更新**: 2026-04-15
- **适用对象**: 采购管理人员、财务人员

## 🚀 快速开始

### 系统要求

- **操作系统**: Windows 10/11, macOS 10.15+, Linux
- **浏览器**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **网络**: 需要互联网连接（部分功能支持离线使用）
- **屏幕分辨率**: 建议1024x768以上

### 首次使用

1. **访问应用**: 打开浏览器，输入应用地址
2. **设备检测**: 系统自动检测设备类型（移动端/桌面端）
3. **界面选择**: 根据设备显示合适的界面布局
4. **开始使用**: 点击"新建方案"开始创建采购方案

## 📱 移动端使用指南

### 首页功能

#### 新建采购方案
1. 点击首页"新建方案"按钮
2. 填写方案基本信息：
   - 方案名称
   - 采购年份
   - 总预算金额
   - 采购人数
3. 系统自动计算人均预算
4. 选择资金来源和采购场景
5. 点击"下一步"继续

#### 商品选择
1. 在商品库中浏览或搜索商品
2. 点击商品添加到采购清单
3. 设置商品数量和规格
4. 系统实时计算金额和合规性

#### 方案预览
1. 查看方案总金额和预算使用情况
2. 检查合规性状态
3. 确认无误后保存方案

### 方案管理

#### 查看历史方案
1. 点击底部导航"历史"
2. 浏览已保存的方案列表
3. 点击方案查看详情

#### 编辑方案
1. 在方案详情页点击"编辑"
2. 修改方案内容
3. 保存修改

#### 删除方案
1. 在方案列表页左滑方案项
2. 点击"删除"按钮
3. 确认删除操作

## 💻 桌面端使用指南

### 双视图功能介绍

#### 快捷编辑视图（默认）
- **三栏布局**: 配置面板 + 商品列表 + 预览面板
- **快速操作**: 支持拖拽、批量操作等
- **实时预览**: 修改即时反映在预览面板

#### 公文排版视图

- **打印优化**: 支持高质量打印输出
- **模板定制**: 可自定义公文模板

#### 视图切换方式
1. **按钮切换**: 点击顶部视图切换按钮
2. **快捷键**: 
   - `Ctrl+1`: 切换到编辑视图
   - `Ctrl+2`: 切换到公文视图

### 高级功能

2. **批量修改**: 选中多个商品后批量设置数量

#### 数据导入导出
1. **导入商品**: 支持Excel格式商品列表导入

## 📊 采购方案创建流程

### 第一步：方案配置

#### 基本信息填写
- **方案名称**: 清晰描述采购目的
- **采购年份**: 选择正确的财务年度
- **总预算**: 输入可用预算总额
- **采购人数**: 参与采购的员工数量

#### 资金配置
- **资金来源**: 
  - 工会经费：适用于节日慰问等
  - 其他资金：适用于其他采购场景
- **人均预算**: 系统自动计算，可手动调整

#### 场景选择
- **节日慰问**: 春节、中秋等节日采购
- **文体活动**: 运动会、文化活动采购
- **其他场景**: 自定义采购需求

### 第二步：商品选择

#### 商品搜索
1. **关键词搜索**: 输入商品名称或分类
2. **分类筛选**: 按食品、日用品等分类筛选
3. **平台筛选**: 筛选832平台商品

#### 商品管理
1. **添加商品**: 点击商品卡片添加到清单
2. **数量设置**: 输入采购数量
3. **价格查看**: 查看商品单价和总价
4. **备注信息**: 添加采购备注

#### 合规检查
- **预算检查**: 实时显示预算使用率
- **价格检查**: 检查商品价格是否合规
- **平台占比**: 显示832平台采购占比

### 第三步：方案生成

#### 预览确认
1. **方案总览**: 查看方案基本信息
2. **商品清单**: 核对采购商品列表
3. **金额统计**: 确认总金额和预算余额
4. **合规状态**: 检查各项合规指标

#### 保存方案
1. **方案命名**: 输入方案保存名称
2. **标签分类**: 添加标签便于查找
3. **保存确认**: 点击保存完成创建

## ❓ 常见问题

### 使用问题

#### Q: 如何创建新的采购方案？
A: 点击首页"新建方案"按钮，按照向导步骤填写信息即可。

#### Q: 为什么商品无法添加到方案？
A: 请检查：1）商品库存状态 2）预算余额 3）合规性限制

#### Q: 如何导出采购方案？
A: 在方案详情页点击"导出"按钮，选择导出格式和配置选项。

### 技术问题

#### Q: 应用加载缓慢怎么办？
A: 1）检查网络连接 2）清理浏览器缓存 3）尝试刷新页面

#### Q: 数据丢失如何恢复？
A: 使用"数据恢复"功能，选择最近的备份文件进行恢复。

#### Q: 兼容哪些浏览器？
A: 支持Chrome、Firefox、Safari、Edge等现代浏览器。

### 合规问题

#### Q: 预算超支如何处理？
A: 系统会提示预算超支，需要调整商品数量或选择价格更低的商品。

#### Q: 832平台占比要求是多少？
A: 根据不同采购场景，832平台占比要求为30%-50%，系统会实时提示。

#### Q: 价格合规标准是什么？
A: 价格合规标准根据采购场景和商品分类设定，系统会自动检查。
