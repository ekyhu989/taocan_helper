## 二、测试范围

### 2.1 按功能模块划分

| 功能模块 | 测试内容 | 优先级 |
|---------|---------|--------|
| **导航与诊断** | 合规预警检测、诊断页展示、一键整改 | P0 |
| **政策文库** | 政策搜索、条款跳转、版本校验、导出拦截 | P0 |
| **公文生成** | 政策引用、禁用词检测、导出排版、签章添加 | P0 |
| **合规测算** | 832占比计算、预警触发、数据锁定、口径切换 | P0 |
| **历史方案** | 多维筛选、标签管理、数据对比、操作日志 | P1 |
| **商品库** | 回收站、批量操作、节日模板、832置顶 | P1 |

| **数据安全** | 云备份、三要素恢复、数据导出、内存清理 | P2 |
| **移动体验** | 大字模式、加载提示、响应式布局 | P1 |

### 2.2 按业务流程划分

1. **采购流程**：Page1 → Page2 → Page3 → 导出
2. **历史管理**：查看历史 → 筛选/搜索 → 复用/删除
3. **商品管理**：查看商品 → 编辑/删除 → 批量操作
4. **合规诊断**：检测风险 → 查看诊断 → 一键整改
5. **数据恢复**：输入三要素 → 查询备份 → 恢复数据
