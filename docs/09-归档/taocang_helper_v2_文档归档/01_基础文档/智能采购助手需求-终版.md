智能采购助手需求规格说明书
1. 项目概述
1.1 项目背景
为满足新疆地区机关单位工会慰问品采购需求，开发一款智能采购助手，帮助用户高效完成采购方案制定、公文生成和合规性检查，提升采购效率并确保政策合规。
1.2 目标用户
• 新疆地区机关单位工会工作人员
• 财务部门采购经办人员
• 单位采购审批人员
1.3 核心价值
• 提供新疆特色采购政策智能引导
• 生成全套采购审批文档
• 实现 832 平台采购比例智能推荐
• 完全本地化处理，保护用户隐私
2. 系统架构
2.1 技术架构
• 前端：微信小程序 + H5 页面
• 后端：轻量级 Node.js 服务
• 数据库：本地缓存 + 设备级行为统计
2.2 部署环境
• 运行环境：微信小程序环境
• 兼容性：支持 iOS、Android 主流微信版本
3. 功能模块设计
3.1 页面架构

页面编号	页面名称	功能描述	访问路径
Page1	基础信息录入	采集采购基础信息	首页入口
Page2	采购方案生成	生成智能采购方案	Page1→Page2
Page3	公文生成与交付	生成全套审批文档	Page2→Page3
Page4	合规测算与政策库	年度合规性测算工具	独立入口
3.2 Page1：基础信息录入页面
3.2.1 页面布局
• 左侧：信息录入表单（占比 70%）
• 右侧：实时方案预览区（占比 30%）
3.2.2 功能板块
1. 基础参数设置 - 采购地区：默认”新疆地区”（不可修改） - 采购场景：下拉选择 - 传统节日慰问（春节、古尔邦节、肉孜节） - 专项活动物资 - 精准帮扶慰问 - 节日类型：联动选择
（根据采购场景动态变化） - 人数：数字输入框（必填） - 总预算：数字输入框（必填） - 资金

来源：下拉选择 - 行政福利费（默认） - 工会经费 - 专项慰问费 - 其他经费 - 预算模式：单选切换 - 按人均标准（默认） - 按总额控制 - 意向品类：多选下拉 - 日用品 - 食品 - 文体用品 - 其他节日礼品
2. 智能引导功能 - 实时计算人均预算：总预算 ÷ 人数 - 新疆政策提示：当人均预算 > 2000 元时显示警告 - 832 平台引导：显示”建议采购比例≥30%“提示
3. 实时预览区 - 预计人均金额：动态显示 - 推荐品类组合：根据选择动态更新 - 示例方案：显示典型采购方案示例
3.3 Page2：采购方案生成页面
3.3.1 页面布局
• 顶部：方案摘要信息栏
• 中部：商品清单表格
• 底部：操作按钮和合规性提示
3.3.2 核心功能
1. 智能商品推荐 - 推荐算法：基于预算、品类、832 比例要求 - 商品来源： - 832 平台商品
（优先推荐） - 本地优质供应商商品 - 历史采购高频商品 - 推荐策略： - 832 商品占比自动控制在 30-50% - 优先推荐新疆本地特色商品 - 确保总价误差≤±5%
2. 商品清单管理 - 表格字段： - 序号 - 商品名称 - 规格 - 单价 - 数量 - 总价 - 来源标识（832 平台/本地供应商） - 操作功能： - 手动添加商品 - 删除商品 - 修改数量 - “换一批”推荐
3. 合规性仪表盘 - 832 平台占比：环形进度条显示 - 状态提示： - ≥30%：绿色对勾”符合要求” - <30%：黄色警告”建议增加”- 年度累计提示：显示”本年度已完成 XX%， 还需采购XX 元”
4. 智能调整功能 - “一键优化”按钮：自动调整商品组合以满足 832 比例要求 - “成本控制”模式：在预算范围内优化商品品质
3.4 Page3：公文生成与交付页面
3.4.1 页面布局
• 左侧：公文配置区
• 右侧：文档预览区
• 底部：操作按钮区
3.4.2 文档包生成
1. 采购申请报告 - 样式要求：极简白底黑字，无花哨边框 - 内容结构： - 标题：关于 202X 年XX 节日慰问品采购的申请报告 - 主送：公司党委/工会委员会 - 正文： - 采购背景（依据新疆总工会 XX 文件） - 采购对象和人数 - 预算标准和总额 - 采购方式说明（优先 832 平台） - 资金来源 - 落款：申请部门、日期 - 智能填充：自动从 Page1 和Page2 获取数据
2. 拟采购物品清单 - 来源：直接引用 Page2 的商品清单 - 格式：标准表格形式 - 内容：序号、商品名称、规格、单价、数量、总价、备注
3. 三方询价记录单 - 功能价值：解决财务报销合规性要求 - 智能填充： - 供应商 A 列：自动填充推荐方案价格 - 供应商 B/C 列：留空供用户填写 - 价格浮动：±5% 随机生成（标注”示例数据”） - 留白设计： - 供应商名称栏 - 实际报价栏 - 询价小组签字栏

4. 工会委员会会议纪要 - 模板类型：填空式模板 - 核心内容： - 会议时间、地点 - 研究事项： XX 节日慰问品采购 - 会议决定：同意采购方案 - 责任部门和经办人
3.4.3 操作功能
• 一键打包下载：生成包含 4 个文档的.zip 文件
• 单独导出：可选择导出特定文档
• 复制文本：支持复制当前预览内容
3.5 Page4：合规测算与政策库
3.5.1 年度进度测算
• 计算逻辑：
‣  输入：年度总预算、已完成采购金额
‣  计算：完成率 = 已完成金额 ÷ 总预算
‣  建议：还需采购金额 = 总预算 × 30% - 已完成金额
• 可视化展示：
‣  进度条：红/黄/绿三色状态
‣  大字号显示完成率
‣  文字提示：是否达标及建议
3.5.2 政策文库
• 文档类型：
‣  新疆维吾尔自治区基层工会经费收支管理办法
‣  关于进一步推进脱贫地区农副产品采购的通知
‣  832 平台采购操作指南
• 访问方式：外部链接跳转或 PDF 在线预览
4. 业务流程设计
4.1 主流程
1. 基础信息录入（Page1）
• 用户输入采购基础信息
• 系统实时生成方案预览
2. 采购方案生成（Page2）
• 系统智能推荐商品组合
• 用户确认或调整方案
3. 公文生成与交付（Page3）
• 用户输入单位信息
• 系统生成全套审批文档
• 用户下载或导出文档
4.2 辅助流程
1. 合规测算（Page4）
• 用户输入年度预算数据
• 系统计算完成进度
2. 政策查询
• 用户查阅相关政策文件
• 系统提供最新政策解读

5. 非功能性需求
5.1 性能要求
• 页面加载时间：≤2 秒
• 方案生成时间：≤3 秒
• 文档生成时间：≤5 秒
5.2 安全要求
• 隐私保护：不收集任何用户个人信息
• 数据安全：所有计算在本地完成
• 合规性：符合政府采购政策要求
5.3 兼容性要求
• 浏览器：微信内置浏览器
• 设备：支持主流手机型号
• 系统：iOS 10+、Android 5.0+
5.4 可用性要求
• 操作简单：3 步完成核心功能
• 引导明确：清晰的操作指引
• 容错性强：输入错误友好提示
