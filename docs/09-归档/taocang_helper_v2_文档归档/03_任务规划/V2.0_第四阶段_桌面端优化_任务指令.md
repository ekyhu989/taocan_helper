### 任务1：双视图切换架构

**任务编号**: DeepSeek-V2.0-005
**优先级**: P1
**预计工时**: 2天
**截止日期**: 2026-04-23

#### 需求描述
实现桌面端双视图切换架构：快捷编辑视图 + 公文排版视图。

#### 具体要求

1. **视图切换组件**:
   ```typescript
   // components/desktop/ViewSwitcher.tsx
   type ViewMode = 'edit' | 'document';

   export const ViewSwitcher: React.FC = () => {
     const [currentView, setCurrentView] = useState<ViewMode>('edit');
     
     return (
       <div className="view-switcher">
         <button 
           className={currentView === 'edit' ? 'active' : ''}
           onClick={() => setCurrentView('edit')}
         >
           快捷编辑
         </button>
         <button 
           className={currentView === 'document' ? 'active' : ''}
           onClick={() => setCurrentView('document')}
         >
           公文排版
         </button>
       </div>
     );
   };
   ```

2. **视图状态管理**:
   - 视图切换状态全局共享
   - 编辑视图修改同步到公文视图
   - 支持快捷键切换（Ctrl+1/Ctrl+2）

3. **路由配置**:
   - `/desktop/edit` - 快捷编辑视图
   - `/desktop/document` - 公文排版视图
   - 支持视图间无缝跳转

#### 验收标准
- [ ] 视图切换组件可用
- [ ] 状态同步正确
- [ ] 快捷键支持
- [ ] 构建通过

#### 文件变更
