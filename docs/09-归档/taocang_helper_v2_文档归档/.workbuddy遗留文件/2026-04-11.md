### 核心流程
1. 接收用户表单数据（单位名称、场景、人数、预算等）
2. 调用 `validateBudget()` 进行预算校验

4. 调用 `assembleReport()` 组装完整报告
5. 返回 `{ title, body, sceneLabel }` 供 UI 展示

## vite命令缺失修复
- **问题**：运行 `npm run dev` 时提示 "vite不是内部或外部命令"
- **诊断**：`node_modules/.bin` 目录下缺少vite可执行文件，但vite包已可通过npx下载（npx vite --version 正常）
- **解决方案**：修改package.json中的npm脚本，添加npx前缀：
  - `"dev": "npx vite"` (原为 `"dev": "vite"`)
  - `"build": "npx vite build"`
  - `"preview": "npx vite preview"`
- **临时效果**：用户可立即启动项目（npx会临时下载vite）
- **长期建议**：运行 `npm install --force` 重新安装所有依赖，恢复本地二进制文件
