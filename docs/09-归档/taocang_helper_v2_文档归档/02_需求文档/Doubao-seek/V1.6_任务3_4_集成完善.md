### 3.4 集成832商品置顶

**修改文件**：`src/components/ProductManager.jsx`

**集成内容**：
1. 商品列表排序时832商品置顶
2. 搜索结果中832商品优先展示
3. 视觉标识强化（绿色标签）

**实现**：
```jsx
// 排序函数
const sortProducts = (products) => {
  return [...products].sort((a, b) => {
    // 832商品置顶
    if (a.isPlatform832 && !b.isPlatform832) return -1;
    if (!a.isPlatform832 && b.isPlatform832) return 1;
    // 然后按修改时间排序
    return new Date(b.updatedAt) - new Date(a.updatedAt);
  });
};

// 商品卡片显示832标识
<div className="product-card">
  {product.isPlatform832 && (
    <span className="badge-832">🟢 832平台</span>
  )}
  {/* 商品信息 */}
</div>
```
