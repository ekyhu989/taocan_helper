### 任务 V1.5-1: 首页（Landing Page）开发
**预计工时**: 6-8小时  
**优先级**: 🔴 高  
**依赖**: 无

#### 功能需求

**页面结构**:
```
HomePage.jsx
├── Header（Logo + 产品名称）
├── HeroSection（项目简介 + 价值主张）
├── FeatureCards（4个功能入口卡片）
│   ├── Card: 制定方案 📝
│   ├── Card: 合规测算 📊
```

**技术要求**:

- [ ] 点击卡片跳转对应功能页面

**交互需求**:
- [ ] 卡片hover效果（阴影、缩放）
- [ ] 点击反馈（颜色变化）
- [ ] 按钮点击动画

### 任务 V1.5-3: 页面切换动画
**预计工时**: 3-4小时  
**优先级**: 🟡 中  
**依赖**: V1.5-1、V1.5-2完成

#### 功能需求

**动画规则**:
| 切换方向 | 动画效果 | 时长 | 缓动 |
|---------|---------|------|------|
| 首页 → 功能页 | 从右向左滑入 | 300ms | ease-in-out |
| 功能页 → 首页 | 从左向右滑出 | 300ms | ease-in-out |
| 功能页之间 | 淡入淡出 | 200ms | ease-in-out |

**技术要求**:
- [ ] 使用CSS Transition或Framer Motion
- [ ] 动画流畅，不卡顿
- [ ] 支持移动端手势（可选）
