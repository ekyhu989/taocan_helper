## 二、详细需求

### 2.1 统一政策引用标准

**当前问题**：
- 不同公文可能引用不同版本的政策文件
- 政策文号格式不统一

**优化方案**：
- 所有公文统一引用最新政策版本
- 政策文号格式标准化

**标准化文号格式**：
```
依据文件：
1. 《新疆维吾尔自治区基层工会经费收支管理办法实施细则》（新工办〔2019〕3号）
2. 《关于做好2025年政府采购脱贫地区农副产品工作的通知》（新财购〔2025〕2号）
3. 《2025年832平台采购全国通知》（财办库〔2024〕267号）
```


### 2.6 敏感内容标准化

**禁用词汇检查**：
```javascript
const forbiddenWords = ['购买', '下单', '支付', '商城', '购物车', '订单'];
const complianceWords = ['采购', '方案', '预算', '物资', '商品库', '申请'];

const validateContent = (content) => {
  const found = forbiddenWords.filter(word => content.includes(word));
  if (found.length > 0) {
    return {
      valid: false,
      message: `发现禁用词汇：${found.join('、')}，请替换为合规术语`
    };
  }
  return { valid: true };
};
```

**自动替换功能**：
- 生成公文前自动检查禁用词汇
- 发现禁用词时提示替换建议
- 关键术语统一标准化

### 3.2 导出前校验

**文件**：`src/utils/reportValidator.ts`

```typescript
export const validateBeforeExport = () => {
  // 1. 政策版本校验
  if (!checkPolicyVersion()) {
    return { valid: false, reason: 'policy_outdated' };
  }
  
  // 2. 禁用词汇校验
  const contentValidation = validateContent(reportContent);
  if (!contentValidation.valid) {
    return { valid: false, reason: 'forbidden_words', message: contentValidation.message };
  }
  
  // 3. 合规性校验
  const complianceValidation = validateCompliance(userInput, productResult);
  if (!complianceValidation.isValid) {
    return { valid: false, reason: 'compliance_risk', risks: complianceValidation.risks };
  }
  
  return { valid: true };
};
```
