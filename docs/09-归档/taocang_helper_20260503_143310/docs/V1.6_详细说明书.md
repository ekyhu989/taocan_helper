### 1.2 预算测算逻辑

#### 1.2.1 人均预算计算

```typescript
// 核心算法
function calculatePerCapitaBudget(totalBudget: number, peopleCount: number): number {
  return Math.floor(totalBudget / peopleCount);
}
```

#### 1.2.2 价格约束机制（V1.6核心优化）

根据**资金来源**设置不同的价格上限约束：

| 资金来源 | 折扣系数 | 价格上限公式 | 示例（预算200元/人） |
|---------|---------|-------------|-------------------|
| **工会经费** | 8折 (0.8) | `上限 = 预算 ÷ 0.8` | 250元 |
| **其他资金** | 9折 (0.9) | `上限 = 预算 ÷ 0.9` | 222元 |

**实现代码**:
```typescript
function calculateMaxPerCapita(budget: number, fundSource: string): number {
  const discountRate = fundSource === '工会经费' ? 0.8 : 0.9;
  return Math.ceil(budget / discountRate);
}
```

**设计意图**:
- 确保生成方案的价格合理性
- 防止预算200元却生成400元套餐的问题
- 给商品选择留出合理空间（考虑品质、品牌等因素）

### 1.4 套餐组合算法

#### 1.4.1 组合逻辑

```typescript
interface SchemeCombination {
  items: Product[];           // 商品列表
  totalPrice: number;         // 总价
  perCapitaPrice: number;     // 人均价格
  categoryCount: number;      // 品类数
  complianceScore: number;    // 合规评分
}

function generateScheme(
  budget: number,
  peopleCount: number,
  maxPerCapita: number,
  products: Product[]
): SchemeCombination {
  // 1. 筛选符合价格区间的商品
  const eligibleProducts = products.filter(p => 
    p.price >= maxPerCapita * 0.5 && 
    p.price <= maxPerCapita
  );
  
  // 2. 按优先级排序
  const sortedProducts = sortByPriority(eligibleProducts);
  
  // 3. 智能组合（背包算法变种）
  const combination = findOptimalCombination(
    sortedProducts,
    budget,
    peopleCount
  );
  
  return combination;
}
```

```typescript
function calculate832Ratio(items: Product[]): number {
  const totalAmount = items.reduce((sum, item) => sum + item.price * item.quantity, 0);
  const amount832 = items
    .filter(item => item.is832Platform)
    .reduce((sum, item) => sum + item.price * item.quantity, 0);
  
  return (amount832 / totalAmount) * 100;
}
```

#### 2.5.1 锁定规则

```typescript
function isDataLocked(date: Date): boolean {
  const currentMonth = date.getMonth() + 1; // 1-12
  const currentDay = date.getDate();
  
  // 12月1日起锁定当年数据
  if (currentMonth === 12) {
    return true;
  }
  
  // 次年1月15日前仍可查看上年数据
  if (currentMonth === 1 && currentDay <= 15) {
    return true;
  }
  
  return false;
}
```
