### 六、核心业务逻辑

#### 1. 832平台占比计算
```
832商品总金额 = sum(item.subtotal for item in items if item.product.is832)
总金额 = sum(item.subtotal for item in items)
832占比 = 832商品总金额 / 总金额
合规阈值：≥30%
```

#### 2. 人均预算校验
- 普通预警：>500元（黄色提醒）
- 新疆政策阈值：>2000元（橙色警告，建议832占比≥30%）

#### 3. 总价误差校验
- 允许范围：≤±5%
- 计算公式：误差率 = |实际金额 - 预期预算| / 预期预算
