## 🚀 部署步骤

### 步骤1: 创建腾讯云账号

1. 访问 [腾讯云官网](https://cloud.tencent.com/)
2. 注册账号并完成实名认证
3. 进入控制台

---

### 步骤2: 开通静态网站托管

#### 方式A: 云开发（推荐，简单）

1. 控制台 → 云开发 → 创建环境
2. 环境名称: `taocang-helper`
3. 选择"按量付费"（有免费额度）
4. 开通静态网站托管

#### 方式B: COS对象存储（更灵活）

1. 控制台 → 对象存储COS → 创建Bucket
2. 名称: `taocang-helper`
3. 地域: 选择离用户最近的（如新疆选"北京"或"成都"）
4. 访问权限: 公有读私有写

---

### 步骤3: 本地构建项目

```bash
# 1. 进入项目目录
cd e:/taocang_helper

# 2. 安装依赖
npm install

# 3. 构建生产版本
npm run build

# 4. 确认dist目录生成
ls dist/
# 应该看到: index.html, assets/, images/ 等
```

---

### 步骤4: 上传文件到腾讯云

#### 方式A: 控制台上传（简单，适合首次）

1. 进入云开发/COS控制台
2. 找到静态网站托管文件管理
3. 点击"上传文件"
4. 选择 `dist` 目录下的所有文件
5. 等待上传完成

#### 方式B: CLI命令行（推荐，适合后续更新）

```bash
# 1. 安装腾讯云CLI
npm install -g @cloudbase/cli

# 2. 登录腾讯云
tcb login

# 3. 初始化项目（首次）
tcb init
# 选择: 云开发环境 → taocang-helper

# 4. 部署（上传dist目录）
tcb hosting:deploy dist -e taocang-helper

# 后续更新只需执行这一步
```

---

### 步骤5: 配置访问

#### 获取访问地址

- **默认域名**: `https://taocang-helper-xxx.tcloudbaseapp.com`
- 复制此地址，在微信中测试访问

#### 配置自定义域名（可选）

1. 购买域名（阿里云/腾讯云/万网）
2. 域名备案（如使用国内服务器）
3. 腾讯云控制台 → 添加自定义域名
4. 配置DNS解析（按腾讯云提示）
5. 申请SSL证书（免费）
6. 等待生效（约10分钟）

## 🔄 后续更新流程

### 方式A: 命令行更新（推荐）

```bash
# 1. 本地修改代码
# 2. 构建
npm run build

# 3. 部署到腾讯云
tcb hosting:deploy dist -e taocang-helper

# 4. 等待约1分钟，线上自动更新
```

1. 重新构建项目 `npm run build`
2. 腾讯云控制台 → 文件管理
3. 删除旧文件，上传新文件
4. 等待CDN刷新（约5分钟）

## ⚠️ 注意事项

### 1. 缓存问题

更新后如未立即生效：
- 清除浏览器缓存
- 等待CDN刷新（5-10分钟）
- 或添加版本号 `?v=1.0.1`

### 2. 文件大小限制

- 单文件最大 500MB
- 本项目远小于限制

### 3. 域名备案

- 使用国内服务器必须备案
- 备案期间可用临时域名访问
- 备案流程约7-20天
