## 🚀 验收后流程

### 如果全部通过
1. 更新版本号至V1.5
2. 构建生产版本 `npm run build`
3. 部署到腾讯云
4. 上线公告

### 如果有条件通过
1. 记录需要修改的问题
2. 分配给对应开发者
3. 修改后重新验收（仅检查修改项）

### 如果不通过
1. 详细说明问题
2. 开发者重新修改
3. 完整重新验收
