# 页面切换动画说明 - V1.5

> 定义首页与功能页之间的切换动画规范
> 设计师：Kimi2.5
> 日期：2026-04-13

---

## 📋 动画概述

页面切换动画采用「滑动过渡」风格，模拟原生App的页面导航体验。动画流畅自然，增强用户的方位感和操作反馈。

### 动画原则
1. **方向一致性**：进入从右向左，退出从左向右
2. **时长适中**：300ms，既流畅又不拖沓
3. **缓动自然**：ease-in-out，符合物理直觉
4. **性能优先**：使用 transform 和 opacity，避免触发重排

## 🎬 动画定义

### 动画1：首页 → 功能页（进入）

**触发条件**：用户从首页点击功能入口（如「制定方案」）

**动画效果**：
```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  当前页面（首页）          新页面（功能页）              │
│  ┌──────────────┐         ┌──────────────┐             │
│  │              │         │              │             │
│  │   首页内容    │    →    │   功能页内容  │  ← 从右侧滑入 │
│  │              │         │              │             │
│  └──────────────┘         └──────────────┘             │
│                                                         │
│  起始状态：                                              │
│  - 首页：translateX(0)，opacity: 1                       │
│  - 功能页：translateX(100%)，opacity: 1（在视口右侧）     │
│                                                         │
│  结束状态：                                              │
│  - 首页：translateX(-30%)，opacity: 0.8（向左淡出）       │
│  - 功能页：translateX(0)，opacity: 1（完全显示）          │
│                                                         │
│  动画参数：                                              │
│  - 时长：300ms                                           │
│  - 缓动：ease-in-out                                     │
│  - 属性：transform, opacity                              │
└─────────────────────────────────────────────────────────┘
```

**CSS实现**：
```css
/* 页面容器 */
.page-container {
  position: relative;
  width: 100%;
  height: 100vh;
  overflow: hidden;
}

/* 页面基础样式 */
.page {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  transition: transform 0.3s ease-in-out, opacity 0.3s ease-in-out;
}

/* 首页 - 进入动画 */
.page-home-enter {
  transform: translateX(-30%);
  opacity: 0.8;
}

.page-home-enter-active {
  transform: translateX(0);
  opacity: 1;
}

/* 功能页 - 进入动画 */
.page-feature-enter {
  transform: translateX(100%);
  opacity: 1;
}

.page-feature-enter-active {
  transform: translateX(0);
  opacity: 1;
}
```

**React + Tailwind实现**：
```jsx
import { useState, useEffect } from 'react';

const PageTransition = ({ children, isEntering, direction }) => {
  const [animationClass, setAnimationClass] = useState('');
  
  useEffect(() => {
    if (isEntering) {
      // 触发进入动画
      requestAnimationFrame(() => {
        setAnimationClass(direction === 'forward' 
          ? 'translate-x-0 opacity-100' 
          : 'translate-x-0 opacity-100'
        );
      });
    }
  }, [isEntering, direction]);
  
  const initialClass = direction === 'forward' 
    ? 'translate-x-full opacity-100' 
    : '-translate-x-full opacity-100';
  
  return (
    <div 
      className={`
        absolute inset-0 w-full h-full
        transition-all duration-300 ease-in-out
        ${initialClass} ${animationClass}
      `}
    >
      {children}
    </div>
  );
};
```

## 📐 动画参数汇总

### 时间参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 动画时长 | 300ms | 页面切换总时长 |
| 延迟 | 0ms | 立即执行，无延迟 |
| 缓动函数 | ease-in-out | 开始和结束较慢，中间较快 |

### 缓动函数定义
```css
/* ease-in-out 等价于 */
cubic-bezier(0.42, 0, 0.58, 1)
```

```javascript
// JavaScript中使用
const easing = 'cubic-bezier(0.42, 0, 0.58, 1)';
```

### 位移参数

| 状态 | 位移值 | 透明度 | 说明 |
|------|--------|--------|------|
| 进入前 | translateX(100%) | 1 | 页面在视口右侧外 |
| 进入后 | translateX(0) | 1 | 页面完全显示 |
| 退出前 | translateX(0) | 1 | 页面当前显示 |
| 退出后 | translateX(100%) | 1 | 页面滑出到右侧 |

### 背景页参数

| 状态 | 位移值 | 透明度 | 说明 |
|------|--------|--------|------|
| 被覆盖时 | translateX(-30%) | 0.8 | 向左偏移并轻微淡出 |
| 恢复时 | translateX(0) | 1 | 恢复正常显示 |

## 🎨 动画细节

### 阴影效果（增强层次感）

在页面切换过程中，添加阴影增强层次感：

```css
.page {
  box-shadow: -4px 0 16px rgba(0,0,0,0.1);
}

/* 进入时阴影变化 */
.page-entering {
  box-shadow: -8px 0 24px rgba(0,0,0,0.15);
}
```

### 底部导航栏处理

底部导航栏**不参与**页面切换动画，保持固定：

```
┌─────────────────────────────────────────────────────────┐
│  ┌─────────────────────────────────────────────────┐   │
│  │              页面内容区域                        │   │
│  │         （参与切换动画）                         │   │
│  └─────────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────────┤  ← 固定
│              底部导航栏（不参与动画）                    │  ← 固定
└─────────────────────────────────────────────────────────┘
```

**实现方式**：
```jsx
<div className="app-container">
  {/* 页面内容区域 - 参与动画 */}
  <div className="page-wrapper relative overflow-hidden">
    <PageTransition key={currentPage}>
      {currentPageComponent}
    </PageTransition>
  </div>
  
  {/* 底部导航栏 - 固定 */}
  <BottomNav className="fixed bottom-0 left-0 right-0 z-50" />
</div>
```

## ⚛️ React实现方案

### 使用 Framer Motion（推荐）

```bash
npm install framer-motion
```

```jsx
import { AnimatePresence, motion } from 'framer-motion';
import { useState } from 'react';

const App = () => {
  const [currentPage, setCurrentPage] = useState('home');
  const [direction, setDirection] = useState(0); // 1: forward, -1: back
  
  const pageVariants = {
    enter: (direction) => ({
      x: direction > 0 ? '100%' : '-100%',
      opacity: 1,
    }),
    center: {
      x: 0,
      opacity: 1,
    },
    exit: (direction) => ({
      x: direction < 0 ? '100%' : '-30%',
      opacity: direction < 0 ? 1 : 0.8,
    }),
  };
  
  const pageTransition = {
    duration: 0.3,
    ease: [0.42, 0, 0.58, 1], // ease-in-out
  };
  
  const navigateTo = (page) => {
    const pageOrder = ['home', 'solution', 'compliance', 'policy', 'products'];
    const currentIndex = pageOrder.indexOf(currentPage);
    const targetIndex = pageOrder.indexOf(page);
    setDirection(targetIndex > currentIndex ? 1 : -1);
    setCurrentPage(page);
  };
  
  return (
    <div className="app">
      <div className="page-container relative h-screen overflow-hidden">
        <AnimatePresence initial={false} custom={direction} mode="wait">
          <motion.div
            key={currentPage}
            custom={direction}
            variants={pageVariants}
            initial="enter"
            animate="center"
            exit="exit"
            transition={pageTransition}
            className="absolute inset-0 w-full h-full bg-white"
          >
            {renderPage(currentPage)}
          </motion.div>
        </AnimatePresence>
      </div>
      
      <BottomNav currentPage={currentPage} onNavigate={navigateTo} />
    </div>
  );
};
```

### 使用纯 CSS + React State

```jsx
import { useState, useEffect } from 'react';

const PageTransition = ({ currentPage, pages }) => {
  const [displayPage, setDisplayPage] = useState(currentPage);
  const [isAnimating, setIsAnimating] = useState(false);
  const [animationDirection, setAnimationDirection] = useState('forward');
  
  useEffect(() => {
    if (currentPage !== displayPage) {
      setIsAnimating(true);
      
      // 判断方向
      const pageOrder = ['home', 'solution', 'compliance'];
      const currentIdx = pageOrder.indexOf(displayPage);
      const targetIdx = pageOrder.indexOf(currentPage);
      setAnimationDirection(targetIdx > currentIdx ? 'forward' : 'back');
      
      // 动画结束后更新显示页面
      const timer = setTimeout(() => {
        setDisplayPage(currentPage);
        setIsAnimating(false);
      }, 300);
      
      return () => clearTimeout(timer);
    }
  }, [currentPage, displayPage]);
  
  const getPageClass = (page) => {
    if (page === displayPage && !isAnimating) {
      return 'translate-x-0 opacity-100';
    }
    if (isAnimating) {
      if (page === displayPage) {
        // 当前页退出
        return animationDirection === 'forward' 
          ? '-translate-x-[30%] opacity-80' 
          : 'translate-x-full opacity-100';
      }
      if (page === currentPage) {
        // 目标页进入
        return animationDirection === 'forward'
          ? 'translate-x-0 opacity-100'
          : '-translate-x-full opacity-100';
      }
    }
    return 'translate-x-full opacity-0';
  };
  
  return (
    <div className="relative w-full h-full overflow-hidden">
      {Object.entries(pages).map(([key, Component]) => (
        <div
          key={key}
          className={`
            absolute inset-0 w-full h-full
            transition-all duration-300 ease-in-out
            ${getPageClass(key)}
          `}
        >
          <Component />
        </div>
      ))}
    </div>
  );
};
```

## 📱 手势支持（可选增强）

### 右滑返回手势

在移动端，支持从屏幕左边缘右滑返回上一页：

```jsx
import { useRef, useCallback } from 'react';

const useSwipeGesture = (onSwipeRight) => {
  const touchStartX = useRef(0);
  const touchStartY = useRef(0);
  
  const onTouchStart = useCallback((e) => {
    touchStartX.current = e.touches[0].clientX;
    touchStartY.current = e.touches[0].clientY;
  }, []);
  
  const onTouchEnd = useCallback((e) => {
    const touchEndX = e.changedTouches[0].clientX;
    const touchEndY = e.changedTouches[0].clientY;
    
    const deltaX = touchEndX - touchStartX.current;
    const deltaY = touchEndY - touchStartY.current;
    
    // 水平滑动距离 > 100px，且垂直偏移 < 50px
    if (deltaX > 100 && Math.abs(deltaY) < 50) {
      // 从屏幕左边缘开始（起始X < 50px）
      if (touchStartX.current < 50) {
        onSwipeRight();
      }
    }
  }, [onSwipeRight]);
  
  return { onTouchStart, onTouchEnd };
};

// 使用
const SwipeablePage = ({ children, onGoBack }) => {
  const swipeHandlers = useSwipeGesture(onGoBack);
  
  return (
    <div 
      className="w-full h-full"
      {...swipeHandlers}
    >
      {children}
    </div>
  );
};
```

## ⚡ 性能优化

### 1. 使用 will-change
```css
.page {
  will-change: transform, opacity;
}

.page-animation-complete {
  will-change: auto;
}
```

### 2. 使用 GPU 加速
```css
.page {
  transform: translateZ(0);
  backface-visibility: hidden;
}
```

### 3. 避免动画期间的重排
- 不要动画 width/height/margin
- 只动画 transform 和 opacity
- 使用 requestAnimationFrame 优化
