## 一、问题描述

### 1.1 现象

**卡片数据与商品清单不一致**:
- **高性价比卡片**: 人均总价 ¥222.00
- **商品清单合计**: 人均合计 ¥164.00

### 1.3 问题定位

**不是显示问题，是算法逻辑缺失！**

| 概念 | 数值 | 说明 |
|------|------|------|
| **人均标准（下限）** | ¥200.00 | 用户输入的预算标准 |
| **折扣系数** | 9折 | 其他经费 |
| **人均上限** | ¥222.22 | 200÷0.9=222.22 |
| **当前商品总价** | ¥164.00 | ❌ **低于下限200！** |

**核心问题**: 算法只考虑了上限约束，**没有下限约束**，导致组合商品总价可能低于用户预算标准。

## 二、正确算法策略

### 2.1 约束条件

```
约束区间: [人均标准, 人均上限]
示例区间: [¥200.00, ¥222.22]

规则:
1. 组合总价 ≥ 人均标准（不低于预算）
2. 组合总价 ≤ 人均上限（不超折扣上限）
3. 目标: 在满足约束的前提下，尽量接近上限（最大化福利）
```

### 2.2 目标函数

```typescript
// 评分逻辑
const score = (totalAmount: number): number => {
  // 不满足约束直接淘汰
  if (totalAmount < perCapitaStandard) return 0;  // 低于下限
  if (totalAmount > perCapitaLimit) return 0;     // 高于上限
  
  // 在区间内，越接近上限越好
  return totalAmount / perCapitaLimit;
};
```

## 五、关键修复代码示例

### 完整算法逻辑框架

```typescript
// src/utils/schemeGenerator.ts

interface GenerateSchemeParams {
  perCapitaStandard: number;  // 人均标准（下限）
  fundingSource: string;      // 资金来源
  personCount: number;        // 人数
  // ... 其他参数
}

interface SchemeConstraints {
  minPerCapita: number;  // = perCapitaStandard
  maxPerCapita: number;  // = perCapitaStandard / discountRate
}

const getDiscountRate = (fundingSource: string): number => {
  switch (fundingSource) {
    case '工会经费': return 0.8;
    case '专项慰问费':
    case '其他资金':
    case '行政经费':
    default:
      return 0.9;
  }
};

const getConstraints = (params: GenerateSchemeParams): SchemeConstraints => {
  const discountRate = getDiscountRate(params.fundingSource);
  
  return {
    minPerCapita: params.perCapitaStandard,  // 下限
    maxPerCapita: params.perCapitaStandard / discountRate  // 上限
  };
};

const isValidCombination = (
  items: SchemeItem[],
  constraints: SchemeConstraints,
  personCount: number
): boolean => {
  const totalAmount = items.reduce(
    (sum, item) => sum + item.product.price * item.quantity, 
    0
  );
  const perCapita = totalAmount / personCount;
  
  // 核心：检查上下限约束
  return perCapita >= constraints.minPerCapita && 
         perCapita <= constraints.maxPerCapita;
};

const calculateScore = (
  items: SchemeItem[],
  constraints: SchemeConstraints,
  personCount: number
): number => {
  if (!isValidCombination(items, constraints, personCount)) {
    return 0;  // 无效组合0分
  }
  
  const totalAmount = items.reduce(
    (sum, item) => sum + item.product.price * item.quantity, 
    0
  );
  const perCapita = totalAmount / personCount;
  
  // 越接近上限越好
  return perCapita / constraints.maxPerCapita;
};
```
