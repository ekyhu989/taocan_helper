## 一、问题描述

### 1.1 现象

**截图显示**：移动端访问"方案"页面显示
```
🚧
开发中
移动端页面正在开发中
[返回首页]
```

### 1.2 根本原因

在 `AppV2.jsx` 第97-107行，移动端非首页页面统一显示"开发中"占位符：

```jsx
{currentPage !== 'home' && (
  <div className="flex items-center justify-center min-h-screen">
    <div className="text-center">
      <div className="text-6xl mb-4">🚧</div>
      <h2 className="text-xl font-bold text-gray-800 mb-2">开发中</h2>
      <p className="text-gray-600">移动端页面正在开发中</p>
      <button onClick={() => handlePageChange('home')} className="mt-4 px-6 py-2 bg-blue-600 text-white rounded-lg">
        返回首页
      </button>
    </div>
  </div>
)}
```

### 1.3 已存在但未使用的移动端组件

| 组件 | 路径 | 说明 |
|------|------|------|
| MobileHomePage | `src/pages/mobile/MobileHomePage.jsx` | 移动端首页 |
| NewScheme | `src/pages/mobile/NewScheme.tsx` | 新建方案页 |
| MobileBottomNav | `src/components/mobile/MobileBottomNav.jsx` | 移动端底部导航 |
