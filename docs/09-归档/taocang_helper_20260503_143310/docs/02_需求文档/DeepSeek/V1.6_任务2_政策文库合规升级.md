## 二、详细需求

**当前问题**：政策链接可能使用本地缓存或第三方链接，存在合规风险

| 832平台采购通知 | https://cg.fupin832.com/... |
| 操作指南 | https://cg.fupin832.com/help/... |

**实现要求**：
- 所有政策链接强制跳转到官方源地址
- 新标签页打开（target="_blank"）
- 禁用本地缓存文件跳转

### 2.2 政策版本每日校验

**功能描述**：
- 系统每日自动检查官网政策是否有更新
- 发现更新时弹出红色强制弹窗
- 未同步最新版本前禁止核心操作（生成公文、导出）

**技术实现**（纯前端模拟）：
```javascript
// 由于纯前端无法真正访问官网API，采用以下方案：
// 1. 在代码中硬编码政策版本号（如 "2025-04-01"）
// 2. 从localStorage读取上次确认的日期
// 3. 如果代码版本 > 确认版本，弹出更新提示
// 4. 用户点击"已确认"后更新localStorage

const POLICY_VERSION = "2025-04-01"; // 代码中硬编码

const checkPolicyVersion = () => {
  const confirmedVersion = localStorage.getItem('policy_confirmed_version');
  if (confirmedVersion !== POLICY_VERSION) {
    showUpdateModal(); // 显示强制更新弹窗
    return false;
  }
  return true;
};
```

**弹窗内容**：
```
⚠️ 政策版本更新提醒

检测到政策文件已更新，请确认查看最新版本：
- 新疆工会经费管理办法（2025年4月修订）
- 832平台采购操作指南（最新版）

[查看更新内容] [我已确认]

注意：未确认前无法生成公文和导出文件
```

## 三、技术实现要点

### 3.1 政策数据结构

**文件**：`src/data/policies.ts`

```typescript
export interface Policy {
  id: string;
  title: string;
  officialUrl: string; // 官方源地址
  version: string; // 版本号（如 "2025-04-01"）
  updateDate: string;
  content: string; // 原文
  interpretation: string; // 解读
  clauses: Clause[]; // 条款
  coreRequirements: string[]; // 核心合规条款
}

export interface Clause {
  number: string; // 如 "第5条"
  title: string;
  content: string;
}
```

### 3.2 版本校验Hook

**文件**：`src/hooks/usePolicyVersion.ts`

```typescript
export const usePolicyVersion = () => {
  const [isVersionValid, setIsVersionValid] = useState(true);
  
  const checkVersion = () => {
    const confirmed = localStorage.getItem('policy_confirmed_version');
    const current = POLICY_VERSION;
    const valid = confirmed === current;
    setIsVersionValid(valid);
    return valid;
  };
  
  const confirmVersion = () => {
    localStorage.setItem('policy_confirmed_version', POLICY_VERSION);
    setIsVersionValid(true);
  };
  
  return { isVersionValid, checkVersion, confirmVersion };
};
```
