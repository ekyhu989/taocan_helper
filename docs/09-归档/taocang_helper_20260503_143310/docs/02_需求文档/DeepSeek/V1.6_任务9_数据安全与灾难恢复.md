## 二、详细需求

### 2.1 腾讯云CloudBase加密备份

**技术方案**：
- 使用腾讯云CloudBase数据库（免费额度内）
- 数据端到端AES加密
- 仅用户本人可解密

**备份内容**：
- 商品库数据
- 历史方案数据
- 用户设置

**备份时机**：
- 手动触发备份
- 重要操作后自动备份（可选）

**实现方式**：
```typescript
// 使用腾讯云CloudBase SDK
import cloudbase from '@cloudbase/js-sdk';

const app = cloudbase.init({
  env: 'taocang-helper-6g9ad4x31a5351d5'
});

const db = app.database();

// 加密备份
const backupData = async (data, key) => {
  const encrypted = AES.encrypt(JSON.stringify(data), key);
  await db.collection('backups').add({
    userId: generateUserId(),
    data: encrypted,
    timestamp: new Date()
  });
};
```

### 2.2 三要素账号恢复

**恢复要素**：
1. 单位全称
2. 采购人姓名
3. 手机号

**恢复流程**：
```
1. 用户输入三要素
2. 系统查询CloudBase数据库
3. 匹配成功显示可恢复的数据列表
4. 用户选择要恢复的备份
5. 解密并恢复到本地
```
