## 🔧 技术实现

### 架构设计
```
CompliancePage (主容器)
├── Tab导航（测算/政策）
├── ComplianceCalculator（测算组件）
│   ├── 数据获取（从localStorage读取单位人数）
│   ├── 输入表单（总预算、已完成金额）
│   ├── 计算逻辑（完成率、还需采购金额、人均金额）
│   ├── 2000元上限校验（橙色警告）
│   ├── 三色进度条可视化
│   └── 数据展示区域
└── PolicyLibrary（政策组件）
    ├── 政策文档列表
    ├── 外部链接跳转
    └── 使用说明
```

### 关键技术点
1. **数据获取**：使用`useEffect`从`localStorage`读取`taocang_solution_form`获取单位人数
2. **实时计算**：使用`useEffect`依赖数组`[totalBudget, completedAmount, headCount]`实现实时计算
3. **2000元校验**：常量`MAX_ANNUAL_PER_CAPITA = 2000`，条件渲染警告组件
4. **条件渲染**：`showAnnualWarning`状态控制警告显示
5. **响应式设计**：Tailwind CSS断点系统，移动端优先

### 核心代码片段
```jsx
// 2000元年度上限校验逻辑
const MAX_ANNUAL_PER_CAPITA = 2000;
const perCapita = headCount > 0 ? total / headCount : 0;
setShowAnnualWarning(perCapita > MAX_ANNUAL_PER_CAPITA);

// 橙色警告组件
{showAnnualWarning && (
  <div className="mb-6 p-4 bg-orange-50 border border-orange-300 rounded-lg">
    <p className="font-bold text-orange-800">⚠️ 合规警告：年度人均预算超过2000元上限</p>
    <p className="text-orange-700">年度人均预算 {perCapita.toFixed(0)} 元，超过 2000 元上限。</p>
  </div>
)}
```
