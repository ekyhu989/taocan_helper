## 前置条件

- [x] Vercel CLI已安装 (`npm install -g vercel`)
- [ ] Vercel账号已注册 (https://vercel.com/signup)
- [ ] 已登录Vercel CLI

---

## 部署步骤

### 步骤1: 登录Vercel

```bash
vercel login
```

- 按提示打开浏览器授权
- 或使用邮箱登录

### 步骤2: 部署项目

```bash
# 进入项目目录
cd e:/taocang_helper

# 部署到生产环境
vercel --prod
```

### 步骤3: 配置项目（首次部署）

首次部署时会提示配置：

```
? Set up and deploy "e:\taocang_helper"? [Y/n] Y
? Which scope do you want to deploy to? [你的用户名]
? Link to existing project? [y/N] N
? What's your project name? [taocang-helper]
? In which directory is your code located? [./]
? Want to override the settings? [y/N] N
```

### 步骤4: 等待部署完成

部署完成后会显示：

```
🔍  Inspect: https://vercel.com/[用户名]/[项目名]/[部署ID]
✅  Production: https://[项目名]-[随机字符].vercel.app
```

## 备选方案

如果Vercel CLI登录有问题，可使用：

### 方案A: GitHub集成（推荐）

1. 访问 https://vercel.com/new
2. 导入GitHub仓库 `ekyhu989/taocan_helper`
3. 选择框架: Vite
4. 构建命令: `npm run build`
5. 输出目录: `dist`
6. 点击 Deploy

### 方案B: Vercel Dashboard

1. 登录 https://vercel.com/dashboard
2. 点击 "Add New Project"
3. 导入GitHub仓库
4. 配置构建设置
5. 部署
