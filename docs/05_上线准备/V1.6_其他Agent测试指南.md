## 四、常见问题

### Q1: Codespaces启动慢？
**A**: 首次启动需要2-3分钟，后续会快很多

### Q2: npm install失败？
**A**: 检查网络连接，或尝试使用 `npm install --registry=https://registry.npmmirror.com`

### Q3: 端口被占用？
**A**: 项目使用Vite默认端口5173，如被占用会自动切换

### Q4: 如何查看日志？
**A**: 在终端运行 `npm run dev`，查看控制台输出
