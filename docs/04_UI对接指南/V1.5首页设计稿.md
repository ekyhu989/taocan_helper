## 🎨 设计概述

### 设计理念
采用「清晰、专业、可信赖」的设计语言，以蓝色为主色调，体现政务工具的严谨性与科技感。首页作为产品门面，需要：
1. **一眼即懂**：用户3秒内理解产品价值
2. **快速入口**：核心功能一键触达
3. **视觉层次**：信息层级清晰，引导自然

### 配色方案
```css
:root {
  --primary: #1890ff;           /* 主色：蓝色 */
  --primary-light: #40a9ff;     /* 主色浅 */
  --primary-dark: #096dd9;      /* 主色深 */
  --success: #52c41a;           /* 成功绿 */
  --warning: #faad14;           /* 警告橙 */
  --error: #ff4d4f;             /* 错误红 */
  --text-primary: #262626;      /* 主要文字 */
  --text-secondary: #595959;    /* 次要文字 */
  --text-tertiary: #8c8c8c;     /* 辅助文字 */
  --bg-primary: #ffffff;        /* 背景白 */
  --bg-secondary: #f5f5f5;      /* 背景灰 */
  --bg-tertiary: #fafafa;       /* 浅灰背景 */
  --border: #d9d9d9;            /* 边框色 */
  --divider: #f0f0f0;           /* 分割线 */
}
```

### 字体规范
- **标题字体**：思源黑体 Bold / system-ui Bold
- **正文字体**：思源黑体 Regular / system-ui
- **字体栈**：`'Source Han Sans CN', 'Noto Sans SC', system-ui, -apple-system, sans-serif`
