# 底部导航栏设计规范 - V1.5

> 全局底部导航组件设计规范，适用于移动端和桌面端
> 设计师：Kimi2.5
> 日期：2026-04-13

---

## 📋 设计概述

底部导航栏是应用的全局导航组件，固定在视口底部，提供4个主要入口：首页、方案、合规、更多。设计遵循「清晰、可触达、状态明确」的原则。

## 🔘 导航项状态

### 状态定义

| 状态 | 图标颜色 | 文字颜色 | 背景 | 说明 |
|------|----------|----------|------|------|
| 默认 | #8c8c8c | #8c8c8c | transparent | 未选中状态 |
| 选中 | #1890ff | #1890ff | transparent | 当前所在页面 |
| 按下 | #1890ff | #1890ff | rgba(24,144,255,0.08) | 点击瞬间 |
| 禁用 | #bfbfbf | #bfbfbf | transparent | 功能不可用 |

```css

  /* 移动端 */

  /* 桌面端 */

    width: 150px;

  }

/* 图标样式 */
.nav-item__icon {
  font-size: 24px;
  color: #8c8c8c;
  transition: all 0.2s ease;
  
  @media (min-width: 1024px) {
    font-size: 28px;
  }

/* 文字样式 */
.nav-item__label {
```

## 📱 交互行为

### 点击反馈
1. **Touch Start**：背景变为 rgba(24,144,255,0.08)，图标和文字变为 #1890ff
2. **Touch End**：背景恢复透明，保持选中状态颜色
3. **动画时长**：200ms ease

### 页面切换
1. 点击导航项后，页面切换动画执行（详见「页面切换动画说明」）
2. 导航栏本身保持固定，不随页面切换动画
3. 新页面加载完成后，对应导航项高亮

### 当前页再次点击
- 如果当前已在选中页面，再次点击同一导航项：
  - 页面滚动到顶部（如果页面有滚动）
  - 或刷新页面数据（如果页面支持刷新）

## 🎯 使用示例

### React组件示例
```jsx
import React, { useState } from 'react';

const BottomNav = ({ currentPath, onNavigate }) => {
  const [moreMenuOpen, setMoreMenuOpen] = useState(false);
  
  const navItems = [
    { icon: '🏠', label: '首页', path: '/home' },
    { icon: '📝', label: '方案', path: '/solution' },
    { icon: '📊', label: '合规', path: '/compliance' },
    { icon: '➕', label: '更多', path: null, isMore: true },
  ];
  
  const moreMenuItems = [
    { icon: '📚', label: '政策文库', desc: '查看工会采购政策文件', path: '/policy' },
    { icon: '📦', label: '商品库', desc: '管理采购商品清单', path: '/products' },
    { icon: '📋', label: '历史方案', desc: '查看已保存的采购方案', path: '/history' },
  ];
  
  const handleNavClick = (item) => {
    if (item.isMore) {
      setMoreMenuOpen(true);
    } else {
      onNavigate(item.path);
    }
  };
  
  return (
    <>
      <nav className="bottom-nav">
        {navItems.map((item) => (
          <button
            key={item.label}
            className={`nav-item ${currentPath === item.path ? 'nav-item--active' : ''}`}
            onClick={() => handleNavClick(item)}
            aria-current={currentPath === item.path ? 'page' : undefined}
          >
            <span className="nav-item__icon">{item.icon}</span>
            <span className="nav-item__label">{item.label}</span>
          </button>
        ))}
      </nav>
      
      {moreMenuOpen && (
        <MoreMenu 
          items={moreMenuItems}
          onClose={() => setMoreMenuOpen(false)}
          onSelect={(path) => {
            setMoreMenuOpen(false);
            onNavigate(path);
          }}
        />
      )}
    </>
  );
};

export default BottomNav;
```
