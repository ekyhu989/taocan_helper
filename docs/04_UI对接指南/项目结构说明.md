## 🎨 现有设计规范提取

### 配色方案（来自src/index.css）

```css
:root {
  --primary-color: #1890ff;        /* 主色调：蓝色 */
  --success-color: #52c41a;        /* 成功：绿色 */
  --warning-color: #faad14;        /* 警告：橙色 */
  --error-color: #ff4d4f;          /* 错误：红色 */
  --text-primary: #333333;         /* 主要文字 */
  --text-secondary: #666666;       /* 次要文字 */
  --bg-primary: #ffffff;           /* 背景白 */
  --bg-secondary: #f5f5f5;         /* 背景灰 */
}
```

### 字体规范

- **标题**: 思源黑体 Bold, 18-24px
- **正文**: 思源黑体 Regular, 14-16px
- **辅助文字**: 12-14px, 灰色

### 间距规范

- **页面边距**: 16px（移动端）/ 24px（桌面端）
- **卡片间距**: 12-16px
- **组件内边距**: 12-16px

## 💡 设计建议

1. **保持现有配色**：以蓝色(#1890ff)为主，与政策文档色调一致
2. **图标简洁**：使用Emoji或线性图标，保持专业感
3. **卡片式设计**：参考现有组件的卡片阴影和圆角
4. **移动端优先**：375px宽度为基准设计
