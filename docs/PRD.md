1.1 产品概述
产品名称： AI采购方案生成工具
产品定位： 仅做方案生成工具，不做电商、不做交易
核心功能： 基于AI技术，为政企单位生成合规的采购申请报告和物资品单
1.2 用户角色
•政企单位采购经办人员
•部门负责人
•财务审核人员
1.3 核心场景
1.传统节日慰问：春节、中秋、端午等全员福利
2.专项活动物资：防暑降温、职工活动、加班慰问
3.精准帮扶慰问：职工生日、生病住院、退休离岗、困难帮扶
1.4 功能需求
1.4.1 基础信息录入
•单位名称（文本输入）
•采购场景（下拉选择：节日福利/职工慰问/简易福利礼品）
•人数（数字输入）
•总预算（数字输入）
•资金来源（文本标签：仅作为文本标签填入报告，不参与逻辑计算）
•申请部门（文本输入）
•申请人（文本输入）
1.4.2 合规性校验
•自动计算人均预算 = 总预算 ÷ 人数
•若人均＞500元，页面黄色小字提醒："当前人均标准较高，建议按单位规定核实调整。"（仅提醒，不阻断）
1.4.3 智能品单生成
•根据人数×人均标准生成建议总额
•系统自动组合商品并计算总价
•基于商品库随机/按场景组合，不做智能凑价，只做简单组合 + 自动核算
•支持方案调整（增删改商品，重新计算合计）
1.4.4 公文生成
•自动生成采购申请报告
•按场景选择对应模板
•自动填充变量内容
•消费帮扶提示：温馨提示：为完成年度消费帮扶任务，建议在食品类采购中优先选用832平台产品，便于集中完成全年指标。
1.5 非功能需求
•界面简洁，操作直观
•响应速度快
•数据安全可靠
•符合政务系统规范
