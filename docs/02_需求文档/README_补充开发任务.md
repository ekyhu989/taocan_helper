## ✅ 补充开发完成后检查清单

- [ ] P1: 年度累计上限2000元校验正确（超支橙色警告）
- [ ] P2: 三方询价单±5%随机数据生成正确
- [ ] P3: 采购地区默认"新疆地区"且只读
- [ ] P3: 节日类型联动显示（仅传统节日慰问场景）
- [ ] P4: Page2年度累计提示显示正确
- [ ] 所有字段值正确传递到公文模板
- [ ] 导出功能包含三方询价单

- [ ] Page1: 单次人均≤500元无预警

- [ ] Page4: 年度累计>2000元显示橙色警告
- [ ] 两个阈值数据独立计算，不冲突

- [ ] Page2的832占比提示引用最新政策文件
- [ ] Page4政策库链接准确且可正常跳转
- [ ] 所有阈值（500/2000/30%/20%/10%）与政策汇编一致
- [ ] 政策文号准确无误

### 集成测试
- [ ] 端到端全流程测试通过
- [ ] 所有阈值场景测试通过
- [ ] 政策内容集成测试通过
