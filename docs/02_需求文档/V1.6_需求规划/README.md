# V1.6 版本需求规划

> 本文档汇总V1.6版本的优化方向和需求建议，供产品决策参考。

---

## 版本定位

**V1.6目标**：在V1.5基础上进行体验优化和功能增强，提升用户操作效率和满意度。

## V1.6 候选功能清单

### P1 - 高优先级（建议V1.6必做）

#### F1: 历史方案管理增强
- 历史方案列表添加搜索功能（按单位名称、日期）
- 添加筛选功能（按场景、按日期范围）
- 批量删除历史方案
- 方案收藏/标记功能

#### F2: 数据导出功能
- 导出历史方案为Excel格式
- 导出采购清单为PDF格式（独立版）
- 导出统计数据报表

#### F3: 性能优化
- 代码分割（Code Splitting）
- 首屏加载优化
- 减少打包体积

### P2 - 中优先级（V1.6可选）

#### F4: 用户体验优化
- 添加页面加载状态（loading spinner）
- 添加空状态提示（无历史方案、无商品等）
- 添加操作成功提示（Toast通知）
- 添加用户反馈入口

#### F5: 使用帮助系统
- 首次使用引导（Onboarding）
- 功能使用说明弹窗
- 快捷键提示

#### F6: 方案对比功能
- 选择2-3个方案进行对比
- 对比维度：预算、人均、832占比、商品种类
- 可视化对比结果

### P3 - 低优先级（V1.7或后续）

#### F7: 主题与个性化
- 深色模式（Dark Mode）
- 主题色切换
- 字体大小调整

#### F8: 多语言支持
- 维吾尔语界面
- 语言切换功能

#### F9: PWA支持
- 离线访问能力
- 添加到主屏幕
- 推送通知

## 需求决策建议

### 推荐V1.6功能范围

**必做（P1）**：
1. ✅ 历史方案搜索/筛选
2. ✅ 数据导出Excel
3. ✅ 代码分割优化

**选做（P2，时间允许）**：
4. 🟡 加载状态/空状态优化
5. 🟡 用户反馈入口

**不做（P3，后续版本）**：
- 多语言、PWA、主题切换

### 理由

1. **历史方案管理**：V1.5已支持保存历史，但随着使用增多，查找会成为痛点
2. **数据导出**：工会工作需要上报数据，Excel是刚需
3. **性能优化**：当前打包体积1.5MB+，需要优化
4. **体验优化**：loading和空状态是细节体验，但影响专业感
