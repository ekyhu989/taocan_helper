# 智能采购助手 V1.5 最终优化需求文档

## 文档基础信息

- 项目名称：智能采购助手

- 产品定位：新疆地区机关单位工会采购合规专用方案生成工具

- 当前版本：V1.5

- 技术栈：React + TypeScript + Tailwind CSS + Vite

- 部署环境：腾讯云云开发 CloudBase

- 原生存储：LocalStorage 纯本地存储、无后端服务

- 适用场景：新疆工会节日慰问采购、专项物资采购、审计合规自查、年度帮扶采购统计

---

## 一、整体现状与核心痛点汇总

1. 交互设计不合理，高频功能收纳至二级菜单，日常操作繁琐；

2. 政策引用存在合规风险，内置缓存文件无法同步官网最新版本，审计易判定违规；

3. 商品库、历史方案功能单一，缺少批量操作、多维筛选、复用合规校验能力；

4. 批量操作无防护机制，易出现手滑误改、金额异常等不可逆问题；

5. 移动端适配不足，小屏展示拥挤、加载卡顿，会议汇报场景体验差；

6. 纯本地存储无自动备份，设备损坏/缓存清理会造成数据永久丢失；

7. 合规预警逻辑单一，部分风险提示精准度不足，易引发审计质疑；

8. 缺少审计汇报配套能力，数据汇总、可视化输出、合规材料导出能力缺失。

---

## 二、分模块详细优化需求

### 1. 导航布局交互优化（最高优先级）

1. 移除底部「更多」折叠菜单，取消二级跳转；

2. 移动端固定5项主导航：首页 / 方案 / 合规 / 政策文库 / 商品库；

3. 桌面端自动扩展导航，新增「历史方案」常驻入口；

4. 新增**合规红色紧急预警按钮**，仅风险项触发闪烁，无风险永久静默；

5. 预警按钮点击直达合规诊断页，展示风险原因、核算逻辑、一键整改方案；

6. 页面切换动画保留，优化动效时长与流畅度。

### 2. 政策文库 合规升级

1. 公文政策文号超链接**强制跳转新疆财政厅/总工会官方源地址**，禁用本地缓存跳转；

2. 系统每日自动校验官网政策版本，检测更新弹出红色强制弹窗，未同步禁止核心操作；

3. 公文导出前强制版本一致性校验，版本不一致直接拦截导出；

4. 政策文库新增**关键词全文搜索、条款编号精准跳转**能力；

5. 官方原文与解读内容分区展示，视觉隔离，避免合规混淆；

6. 导出公文尾部自动加注「政策版本校验签章」，包含校验时间、文件来源；

7. 核心合规条款高亮置顶展示（832占比≥30%、年度人均≤2000元等）。

### 3. 商品库管理功能升级

1. 优化排序逻辑：默认按最近修改排序，适配高频编辑场景；

2. 新增商品回收站（7天保留）+ 永久操作日志，支持误删一键恢复；

3. 增加**批量修改防护**：批量改价/改数量强制二次确认，展示修改数量、总金额变动；

4. 金额变动阈值预警：单价变动超50%黄色提醒、超100%强制二次确认；

5. 新增节日模板库：预置春节、古尔邦节、肉孜节标准化商品礼包，一键加载；

6. 832标识商品列表/搜索结果置顶展示，强化帮扶采购引导；

7. 预留批量导入/CSV导出功能迭代接口。

### 4. 历史方案 全面升级

1. 新增多维筛选：支持按节日、年份、预算区间、832占比、商品组合、供应商筛选；

2. 支持自定义标签管理，方案保存可手动打标，标签关键词检索；

3. 历史方案加载**强制全量合规校验**，超标、占比不达标自动拦截；

4. 复用方案新增**数据差异高亮对比**，结合当前人数/预算自动给出调整建议；

5. 所有方案复用、修改操作留痕，日志可导出用于审计追溯；

6. 列表页新增核心数据速览：总金额、832占比、人均金额、合规状态；

7. 优化懒加载机制，压缩加载耗时至 0.1s 内，解决移动端卡顿。

### 5. 公文生成 防错合规优化

1. 统一政策引用标准，全部对接官方最新文件；

2. 禁止过期政策文件生成公文，系统强制拦截；

3. 优化打印样式、Word/PDF 导出排版，适配机关单位公文规范；

4. 敏感内容标准化输出，统一禁用违规词汇，规范采购术语。

### 6. 批量操作 安全防护体系

1. 批量操作后新增**30秒全局一键撤销**按钮，快速回滚误操作；

2. 全量操作日志永久留存：操作时间、修改前后数据、金额变动明细；

3. 前端数据二次校验，拦截极端异常数值录入；

4. 所有高危操作增加弹窗文字确认，降低误触概率。

### 7. 移动端体验专项优化

1. 系统设置新增**大字模式**，金额、单价、核心合规数据加粗放大展示；

2. 精简页面信息密度，优化手机端自适应布局；

3. 所有异步操作增加加载状态提示，避免用户误以为卡死；

4. 优化长列表、复杂数据页面渲染性能，适配低配置手机。

### 8. 合规测算 规则精细化

1. 832占比默认采用新疆官方要求：**金额占比核算**，设置可切换入口并标注审计口径；

2. 单次人均500元黄色预警、年度人均2000元橙色强预警逻辑固化；

3. 手动录入数据与系统自动核算数据视觉区分，标注来源防止人为篡改；

4. 年度数据支持年末锁定只读，杜绝后期修改规避合规要求。

### 9. 数据安全与灾难恢复

1. 摒弃单一本地存储，新增**微信云开发端到端AES加密自动备份**；

2. 实现三要素账号恢复：单位全称+采购人姓名+手机号，一键恢复全量数据；

3. 新增单位名称模糊匹配算法，解决简称/全称匹配失败问题；

4. 兜底方案：支持单位证件+人工审核恢复数据；

5. 支持全量数据手动导出JSON/Excel，本地留存备份；

6. 页面关闭/数据导出后自动清理内存，防止敏感数据泄露。

## 三、新增彩蛋功能：审计汇报一键生成PPT

### 功能名称

审计防哭脸 · 一键生成合规汇报PPT（V1.7 迭代）

### 核心需求

1. 首页新增独立功能按钮，一键自动生成标准化政务PPT；

2. PPT固定结构：封面、年度合规总览、采购明细、风险整改闭环、操作日志摘要；

3. 自动生成可视化图表：合规率环形图、832采购占比柱状图、年度预算仪表盘；

4. 技术强制规避风险：

    - 配置 `html2canvas` 跨域代理、高清渲染参数；

    - ECharts 图表预渲染为高清PNG，防止画布变形空白；

    - 分块分页生成，解决移动端大文件卡死问题；

    - 导出前自动敏感数据脱敏，清除内部标识与隐私信息；

5. 导出规范：

    - 文件名自动携带时间：`年月日_单位_工会采购审计汇报.pptx`；

    - 图表、文字全程可编辑，非纯图片锁定；

    - 移动端生成时长控制在10秒以内，超时人性化提示；

6. 代码内置专属彩蛋注释，保留项目定制标识。

## 四、技术强制约束与风险规避方案

1. 全程无用户隐私明文收集，所有云端数据加密存储，仅本人可解密；

2. 不新增后端服务，保持轻量化、离线可用、内网适配特性；

3. 所有跨域资源开启CORS代理，杜绝截图白屏、资源加载失败；

4. 异步操作增加延时等待，确保图表、接口、动态数据渲染完成；

5. 定时任务轻量化，政策版本校验采用低频次后台轮询；

6. 所有新增功能兼容现有V1.5数据结构，无缝升级、不覆盖旧数据。
