## 二、详细需求

### 2.1 832占比核算口径

**默认口径**：金额占比（新疆官方要求）

**计算公式**：
```
832占比 = 832平台商品金额 ÷ 采购总金额 × 100%
```

**切换入口**（可选）：
- 设置中可切换为"数量占比"
- 切换时标注审计口径

**UI设计**：
```
合规测算页面:
┌─────────────────────────────────────────┐
│ 832平台采购占比                          │
│ 核算口径: [金额占比▼] *新疆审计标准       │
│                                         │
│ 35% ████████████░░░░                    │
│ 达标 ✓ (要求≥30%)                       │
└─────────────────────────────────────────┘
```

### 2.2 预警逻辑固化

**预警规则**：
| 预警类型 | 触发条件 | 响应 |
|---------|---------|------|
| 单次预警 | 人均 > 500元 | 黄色提示 |
| 年度强预警 | 人均 > 2000元 | 橙色警告+拦截 |
| 832不达标 | 占比 < 30% | 红色警告 |

**预警样式**：
- 黄色提示：背景#FFFBE6，文字#D48806
- 橙色警告：背景#FFF2E8，文字#FA541C
- 红色警告：背景#FFF1F0，文字#CF1322

### 2.3 数据来源区分

**视觉区分**：
- 系统自动核算数据：黑色文字
- 手动录入数据：蓝色文字+"手动"标签

**实现方式**：
```typescript
interface DataField {
  value: number;
  source: 'auto' | 'manual';
}

// UI展示
<span className={source === 'manual' ? 'text-blue-600' : 'text-gray-900'}>
  {value}
  {source === 'manual' && <Tag size="sm">手动</Tag>}
</span>
```

### 2.4 年末数据锁定

**功能描述**：
- 年度数据支持年末锁定
- 锁定后只读，不可修改
- 防止后期修改规避合规

**锁定规则**：
- 手动触发锁定
- 锁定后显示锁定时间和操作人
- 解锁需二次确认

**UI设计**：
```
┌─────────────────────────────────────────┐
│ 2025年度合规测算          [已锁定🔒]     │
│ 锁定时间: 2026-01-15 18:30              │
│ 操作人: 管理员                          │
├─────────────────────────────────────────┤
│ 数据只读，不可修改                      │
│ [申请解锁]                              │
└─────────────────────────────────────────┘
```
