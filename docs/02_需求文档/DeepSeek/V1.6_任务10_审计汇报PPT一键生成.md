## 二、详细需求

### 2.1 PPT固定结构

**幻灯片结构**：
1. **封面**
   - 单位名称
   - 汇报标题：2025年度工会采购合规汇报
   - 汇报时间
   - 汇报人

2. **年度合规总览**
   - 年度预算总额
   - 已完成采购金额
   - 合规率
   - 风险项数量

3. **采购明细**
   - 按节日分类统计
   - 按品类分类统计
   - 832平台采购占比

4. **风险整改闭环**
   - 发现的风险项
   - 整改措施
   - 整改结果

5. **操作日志摘要**
   - 关键操作统计
   - 合规检查记录

### 2.2 可视化图表

**图表类型**：
1. **合规率环形图**
   - 合规 / 警告 / 超标 占比

2. **832采购占比柱状图**
   - 按月份/季度统计

3. **年度预算仪表盘**
   - 已使用 / 剩余预算

**技术实现**：
```typescript
// 使用ECharts生成图表
import * as echarts from 'echarts';

// 预渲染为PNG
const renderChartToPNG = (chartOption) => {
  const chart = echarts.init(document.createElement('div'));
  chart.setOption(chartOption);
  return chart.getDataURL({ type: 'png', pixelRatio: 2 });
};
```

### 2.3 PPT导出规范

**文件名格式**：
```
{年月日}_{单位}_工会采购审计汇报.pptx
例如：20250413_新疆某单位_工会采购审计汇报.pptx
```

**技术要求**：
- 使用pptxgenjs库生成
- 图表预渲染为高清PNG（防止变形）
- 文字可编辑（非纯图片）
- 分块分页生成（防卡死）

**导出前处理**：
- 敏感数据脱敏
- 清除内部标识
- 隐私信息过滤

### 2.4 性能要求

**生成时长**：
- 移动端控制在10秒以内
- 超时显示人性化提示

**优化措施**：

### 3.2 图表预渲染

**使用html2canvas + ECharts**：
```typescript
import html2canvas from 'html2canvas';

const captureChart = async (chartRef) => {
  const canvas = await html2canvas(chartRef, {
    scale: 2, // 高清渲染
    useCORS: true,
    logging: false
  });
  return canvas.toDataURL('image/png');
};
```
