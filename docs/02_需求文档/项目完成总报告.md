## 一、项目概述

AI采购方案生成工具是一个为政企单位设计的合规采购方案生成系统。项目目标是开发一个功能完整、界面友好、符合政府采购规范的Web应用，帮助用户快速生成合规的采购申请报告和物资品单。

### 项目特点：
- ✅ **合规性优先**: 严格遵循政府采购术语规范，避免"购买/下单/支付"等电商词汇
- ✅ **智能化生成**: 基于AI技术智能生成采购方案和公文
- ✅ **多格式导出**: 支持Word、PDF格式导出，支持直接打印
- ✅ **移动友好**: 响应式设计，适配手机、平板、桌面端
- ✅ **数据安全**: 本地数据持久化，保护用户隐私

## 五、技术架构回顾

### 前端技术栈
- **框架**: React 18 + Vite + TypeScript
- **样式**: Tailwind CSS + 自定义CSS
- **状态管理**: React Hooks
- **构建工具**: Vite

### 核心模块
1. **预算校验模块** (`budgetValidator.ts`): 预算合法性校验与中文大写金额转换
2. **品单生成模块** (`productListGenerator.ts`): 智能品单生成算法
3. **报告组装模块** (`reportAssembler.ts`): 公文模板填充与组装
4. **导出工具模块** (`exportUtils.ts`): Word/PDF文档生成
5. **数据存储模块**: 表单存储、历史存储、商品存储

### 组件架构
- **基础信息表单** (`BasicInfoForm.jsx`): 6字段基础信息录入
- **方案预览组件** (`SolutionPreview.jsx`): 品单展示与手动调整
- **公文预览组件** (`ProcurementReport.jsx`): 公文展示与导出
- **商品管理组件** (`ProductManager.tsx`): 商品库管理界面
- **历史管理组件** (`HistoryManager.jsx`): 历史方案管理
