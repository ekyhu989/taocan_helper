## 任务概述

在现有的方案预览页面（SolutionPreview）中增加商品手动调整功能，允许用户增删改商品、调整数量，并实时重新计算金额和校验合规性。

---

## 验收标准（必须全部完成）

### 功能实现

- [x] 点击"编辑"弹出数量输入框，可修改数量，实时更新小计金额
- [x] 点击"删除"移除该商品，剩余商品自动重新分配预算（按原算法逻辑）

- [x] 商品被手动调整后，在方案摘要区域显示"已手动调整"标识
- [x] 调整后若人均预算超过500元，在页面显示黄色合规提醒

### 数据流设计

```
用户操作 → App.jsx状态更新 → SolutionPreview重新渲染

```

### 关键代码片段示例

```typescript
// 数量修改处理
const handleQuantityChange = (productId: string, newQuantity: number) => {
  const updatedItems = items.map(item => 
    item.product.id === productId 
      ? { ...item, quantity: newQuantity, subtotal: item.product.price * newQuantity }
      : item
  );
  onAdjustProduct(updatedItems);
};

// 删除商品处理
const handleRemoveProduct = (productId: string) => {
  if (confirm('确定要删除此商品吗？')) {
    const filteredItems = items.filter(item => item.product.id !== productId);
    // 重新分配剩余预算
    onRemoveProduct(filteredItems);
  }
};
```
