# 任务 P5: 政策内容系统集成

## 任务概述

**任务编号**: P5  
**任务名称**: 政策内容系统集成  
**分配给**: GLM-5.1  
**预计工时**: 2-3小时  
**优先级**: 高  
**前置依赖**: P1(Page4合规测算) 必须完成

**任务目标**:  
将《新疆工会采购政策汇编（2025版）》中的政策内容集成到系统各页面，确保公文模板、提示文案、政策库链接与最新政策保持一致。

---

## 背景说明

政策汇编文档位置: `docs/01_基础文档/新疆工会采购政策汇编（2025版）.md`

该文档包含以下关键内容需要集成到系统：
1. **政策文号**: 新工办〔2019〕3号、新财购〔2025〕2号、财办库〔2024〕267号
2. **合规阈值**: 500元单次预警、2000元年度上限
3. **节日范围**: 古尔邦节、肉孜节（新疆特色）
4. **官方入口**: 财政厅、832平台等链接
5. **合规红线**: 八不准、禁用项说明

## 技术实现细节

### 公文模板变量更新

在 `src/reportAssembler.ts` 中添加/更新变量：

```typescript
// 新增政策相关变量
interface ReportVariables {
  // 原有变量...
  
  // 新增政策变量
  policyDocNo: string;        // 政策文号：新财购〔2025〕2号
  policyDocName: string;      // 政策名称
  platformRatio: number;      // 832平台占比
  complianceStatus: string;   // 合规状态说明
}

// 模板示例
const reportTemplate = `
关于{{year}}年{{festival}}慰问品采购的申请报告

公司党委/工会委员会：

根据《新疆维吾尔自治区基层工会经费收支管理办法实施细则》（新工办〔2019〕3号）
及《关于做好2025年政府采购脱贫地区农副产品工作的通知》（{{policyDocNo}}）文件要求，...

本次采购优先通过832平台采购脱贫地区农副产品，占比{{platformRatio}}%，符合≥30%的要求。
`;
```

- [ ] Page1的500元预警文案明确提及"年度上限2000元"
