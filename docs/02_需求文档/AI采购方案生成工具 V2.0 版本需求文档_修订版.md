# AI采购方案生成工具 V2.0 版本需求文档（修订版）

**文档版本**：V1.1（修订版）

**编制角色**：产品核心开发者

**适用范围**：工会采购合规、慰问方案编制、年度台账管理

**核心背景**：基于V1.6全量用户痛点、一线工会干事真实办公场景、审计合规要求、移动端现场刚需迭代升级；坚守工具**免费开源**底层定位，保留客服引流商业逻辑，不无限度开放个性化功能、不破坏产品轻量化架构。

**修订说明**：
- 移除电子盖章功能（不符合实际流程）
- 整合V1.7版本需求
- 优化部分表述

## 一、产品整体战略定位

1. **核心定位**：基层工会采购智能合规工具，减负+防审计风险为第一目标

2. **终端战略**：**移动端优先、桌面端辅助**

    - 移动端：独立专属重构，放弃V1.6挤压式响应式，打造工业级现场作业工具

    - 桌面端：保留专业输出能力，仅弱化复杂编辑、强化公文排版/打印/归档

3. **设计核心原则**

    - 一切功能贴合：车间/慰问现场/会议临时调整/审计现场核查 真实场景

    - 合规透明化：所有计算逻辑、约束规则、修改痕迹**可看、可查、可导出、可审计**

    - 极简不臃肿：拒绝过度个性化，控制功能复杂度，保证长期稳定易用

4. **不可突破底线**

    - 商品公共库不开放自定义新增，永久入库商品需客服审核对接

    - 纯本地存储为主，无强制云端、无复杂权限审批体系

    - 保留客服触点，维持免费工具运营可持续性

---

## 二、整体架构设计

### 2.1 双端架构模式

1. **逻辑层**：双端100%共用（测算规则、合规引擎、商品算法、数据计算）

2. **数据层**：本地`localStorage`统一规范，年度物理隔离

3. **渲染层**：双端差异化UI

    - 移动端：卡片流、大按钮、单手热区、高对比度专属界面

    - 桌面端：双视图切换「快捷编辑视图+公文排版视图」

### 2.2 数据底层规范（强制落地）

1. 所有年度数据存储键名**带年份后缀**，实现物理级隔离

    示例：`annualBudget_2026`、`annual832Total_2027`

2. 方案数据、修改日志、锁定记录与方案绑定存储，不独立零散缓存

3. 本地缓存清理、浏览器重置不丢失档案：**日志&修改痕迹随导出文件固化留存**

## 三、定制方案生成模块 迭代需求（核心优化）

### 3.1 首页与新建方案入口优化

1. 首页中心增加**超大悬浮「+ 新建采购方案」固定按钮**，无隐藏、无小卡片深埋

2. 首页顶部增设场景化快捷入口：春节慰问/中秋慰问/专项活动/困难帮扶

3. 场景选择后自动预设：资金来源、合规预算约束、场景合规提示

### 3.2 方案初始化首屏布局

1. 首屏全量展示核心配置，取消折叠/高级设置隐藏

2. 必填项直接展示：总预算、人数、**资金来源（工会经费/其他资金）**

3. 资金来源增加合规高亮说明：明确8折/9折价格约束、结算差异业务依据

4. 杜绝核心合规因子隐藏，从源头规避审计填报错误

### 3.3 价格约束&商品匹配逻辑透明化

1. 取消算法黑盒，新增**合规测算明细可展开面板**

    展示：人均预算计算公式、资金来源折扣系数、单品价格上限、商品匹配区间

2. 商品选购页常驻价格上限提示条，手动加购超限展示：超限具体金额+上限数值

3. 价格约束模块增加「合规政策依据」弹窗，展示法规条款+供应商折扣业务说明

4. 系统自动计算全程留痕，支持审计直接解读

### 3.4 商品库与临时商品管理

1. 保留V1.6核心规则：公共预设库仅官方审核商品，永久入库需联系客服

2. 新增**临时商品备注功能**（轻量化、不破坏商业逻辑）

    - 仅生效于当前方案，不进入公共商品库

    - 自动生成合规标签：待客服核验/已832资质核验+工单号

    - 支持832平台链接一键导入，自动抓取商品信息，减少手动录入错误

3. 移动端内置快捷客服入口，无需跨APP跳转，保留客服引流触点

### 3.5 重新生成功能全面升级

1. 增加**防误触二次确认**：检测到用户手动修改商品/数量时，强弹窗警示数据清空风险

2. 自动缓存上一版方案，新增「恢复上一版」一键回退功能

3. 重新生成后自动弹出**方案对比视图**，并排展示：总价、品类数、832占比、人均价格

4. 多版本优劣自动标色提示，满足领导对比汇报场景

### 3.6 方案保存与档案管理

1. 自动生成规范方案名称：年份+场景+资金来源+人均档位，支持一键直接保存

2. 自动绑定多维度标签：年份/场景/资金类型/预算档位，支持筛选检索

3. 历史方案列表支持：年份筛选、场景筛选、标签筛选，告别纯时间乱序排列

4. 新增重名保护：同名方案自动提醒，支持自动加副本后缀，禁止静默覆盖

5. 移动端简化输入，以点选、标签选择为主，减少手动打字操作

## 四、合规管理模块 迭代需求（核心优化）

### 4.1 832占比核算口径优化

1. 口径主次强制区分

    - 默认锁定：**按金额（审计唯一主口径）**，高亮政策标注

    - 按数量：降为辅助参考口径，标注「内部统计使用，不做审计依据」

2. 差异化阈值配置

    - 金额口径：固定30%政策红线

    - 数量口径：下调参考阈值，适配慰问品单价差异现实

3. 导出报告仅展示主口径数据，杜绝双数据冲突，辅助口径仅页面内参考

### 4.2 合规红色预警阻断优化

1. 取消单纯"拦截+仅知道了按钮"的断头设计

2. 红警触发后提供闭环修复引导：

    - 一键返回方案编辑页

    - 快速筛选替换832合规商品

    - 一键删减非832商品

3. 增加合规例外备注机制：政策兜底场景可填写说明，导出自动附带例外说明文件

### 4.3 年度手动数据录入追溯体系

1. 手动录入数据视觉强化：单元格黄底高亮、悬浮展示修改批注

2. 全操作自动留痕：修改时间、前后数值、操作行为完整记录

3. 新增数据一键锁定/解锁功能，锁定后禁止编辑，解锁修改全流程日志记录

4. 导出固化日志：

    - Excel：新增隐藏「修改日志」Sheet页

    - PDF：文档末尾附加数据修改日志附录

5. 异常数据强校验：禁止负数、禁止已使用预算超年度总预算，实时预警

### 4.4 年末锁定机制&年度数据隔离

1. 锁定范围解耦：仅锁定**当年已保存历史台账/方案**为只读，不限制新年份创建

2. 年份选择器全年开放，无年末灰化锁定，支持12月提前编制下一年度方案

3. 跨年方案归属规则：方案绑定所选年份，数据、832占比、预算累计独立核算

4. 复制历史方案规则：仅复制商品结构/配置，**不携带年度累计数据**，新年份数据从零计算

5. 设备时间容错：读取本地时间，时间异常仅提示、不限制功能使用

## 五、移动端V2.0 独立专属设计要求（现场工业级标准）

### 5.1 户外强光可视性标准

1. 强制高对比度配色：纯黑正文、纯白底色、高饱和预警色，废除浅灰小字/淡色元素

2. 字号强制底线：辅助文字≥16px，核心数据/按钮文字≥18px加粗

3. 所有合规说明、标签、测算数据**默认直显**，无折叠、无长按查看

### 5.2 单手操作热区规范

1. 屏幕下半40%为专属单手热区，**所有高频核心按钮全部下沉**

2. 顶部导航仅保留返回功能，重新生成、保存、合规检查、添加商品统一底部固定

3. 按钮尺寸最小48px×48px，悬浮主按钮直径≥60px，适配手指操作、降低误触

### 5.3 多通道操作反馈

1. 新增设备振动反馈机制（强制生效）

    - 短振1次：操作成功（保存/生成/确认）

    - 长振2次：合规拦截/数据错误/价格超限

    - 间歇短振：二次确认、待操作提醒

2. 适配嘈杂现场：脱离提示音依赖，支持盲操判断操作结果

### 5.4 现场应急模式

1. 10秒快速创建草稿：仅填写预算、人数、场景，自动生成极简方案

2. 应急草稿自动存为**未完成正式方案**，纳入历史列表，支持跨设备、跨场景续编

3. 草稿转正无缝衔接：办公室打开直接编辑，完整保留现场所有填报数据

4. 支持离线缓存，无网络环境可正常编辑查看

## 六、桌面端功能保留与定位

1. 定位：**专业输出辅助端**，不弱化公文核心能力

2. 双视图切换

    - 快捷编辑视图：适配移动端操作逻辑，快速调整方案

    - **公文排版视图**：独立优化，支持页边距、字体、行距、红头模板

    - ~~电子签章~~（已移除）：实际流程需物理盖章，系统仅提供标准排版模板

3. 完整导出能力：PDF/Excel高精度导出，适配单位打印盖章、归档、审计上报

4. 批量管理功能：历史方案批量查看、批量导出、台账汇总整理

## 七、安全、合规与追溯体系

1. 全链路留痕：数据修改、锁定解锁、方案复制、临时商品审核，全部自动日志记录

2. 日志双备份：本地存储+导出文件固化，杜绝缓存清理导致痕迹丢失

3. 合规标签体系：临时商品、客服核验商品、手动修改数据全量显性标注

4. 审计闭环：所有测算公式、约束依据、修改记录、832核验信息，可直接打印归档

---

## 八、验收标准（硬性上线门槛）

1. 户外场景：正午阳光直射下，所有文字、按钮、数据100%清晰可视

2. 单手操作：单手握机、手提物资状态下，可独立完成新建-编辑-合规-保存全流程

3. 数据隔离：跨年无数据串扰，年度台账独立，复制方案不污染累计预算

4. 合规无漏洞：832口径清晰、手动修改可追溯、红警修复路径完整

5. 双端协同：移动端草稿可在桌面端无缝编辑、排版、导出，工作流无断点

---

## 九、版本迭代约束

1. 优先落地：移动端独立重构 > 合规逻辑优化 > 桌面端排版升级

2. 保持工具轻量化，不新增复杂审批、云端账号、多级权限等重功能

3. 所有优化围绕一线真实痛点，拒绝UI美观优先、脱离现场的设计方案
