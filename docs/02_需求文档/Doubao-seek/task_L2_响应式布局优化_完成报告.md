## 四、技术实现细节

### 断点设计
| 断点 | 宽度 | 布局调整 |
|------|------|----------|
| sm | 640px | 手机/小屏，单列布局 |
| md | 768px | 平板，双列布局开始 |
| lg | 1024px | 小桌面，完整双列 |
| xl | 1280px | 大桌面，保持现有 |

### 关键CSS类
```css
/* 响应式网格 */
grid-cols-1 md:grid-cols-2

/* 响应式间距 */
gap-3 md:gap-4

/* 响应式内边距 */
p-4 md:p-6

/* 响应式字号 */
text-xl md:text-2xl

/* 移动端隐藏文字 */
hidden sm:inline

/* 按钮最小高度 */
min-h-[44px]
```

### 移动端优化策略
1. **单列布局**：在小屏幕上，所有表单字段垂直排列
2. **全宽按钮**：在小屏幕上，按钮占满宽度便于点击
3. **精简文字**：在小屏幕上，只显示图标或简短文字
4. **触摸友好**：所有交互元素至少44x44px
5. **表格滚动**：表格支持横向滑动查看完整内容
