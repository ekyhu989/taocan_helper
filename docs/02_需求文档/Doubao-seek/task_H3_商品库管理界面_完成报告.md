## 八、使用说明

### 8.1 打开商品库管理
点击应用顶部导航栏右侧的"📦 商品库管理"紫色按钮。

### 8.2 新增商品
1. 点击"新增商品"按钮
2. 填写商品信息（带*号为必填项）
3. 点击"确认保存"

### 8.3 编辑商品
1. 在商品列表中找到要编辑的商品
2. 点击"编辑"按钮
3. 修改商品信息
4. 点击"确认保存"

### 8.4 删除商品
1. 在商品列表中找到要删除的商品
2. 点击"删除"按钮
3. 在确认对话框中点击"确认删除"

### 8.5 搜索和筛选
- 在搜索框输入商品名称进行搜索
- 使用分类下拉框筛选特定分类的商品

### 8.6 恢复默认商品库
1. 点击"⚠️ 恢复默认商品库"按钮
2. 在确认对话框中点击"确认恢复"
3. 所有自定义修改将丢失，恢复到初始状态
