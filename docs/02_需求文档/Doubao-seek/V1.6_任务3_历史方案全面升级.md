### 2.3 强制全量合规校验

**触发时机**：
- 历史方案列表加载时
- 点击"使用此方案"时
- 方案详情页打开时

**校验内容**：
```javascript
const validateCompliance = (scheme) => {
  const risks = [];
  
  // 1. 年度人均校验
  const perCapita = scheme.totalBudget / scheme.headCount;
  if (perCapita > 2000) {
    risks.push({ type: 'annual_limit', message: '年度人均超标' });
  }
  
  // 2. 单次预警校验
  if (perCapita > 500) {
    risks.push({ type: 'single_warning', message: '单次人均超过预警线' });
  }
  
  // 3. 832占比校验
  const rate = scheme.platform832Amount / scheme.totalAmount;
  if (rate < 0.3) {
    risks.push({ type: 'platform832', message: '832占比不达标' });
  }
  
  return {
    isValid: risks.length === 0,
    risks
  };
};
```

**拦截逻辑**：
- 校验不通过时，"使用此方案"按钮禁用或显示警告
- 点击后弹出风险提示弹窗
- 用户确认"强制使用"后才可继续

### 3.1 筛选逻辑

**文件**：`src/utils/historyFilter.ts`

```typescript
export const filterHistory = (
  history: HistoryItem[],
  filters: {
    festival?: string;
    year?: number;
    budgetRange?: [number, number];
    platform832Range?: [number, number];
    tags?: string[];
  }
) => {
  return history.filter(item => {
    if (filters.festival && item.festival !== filters.festival) return false;
    if (filters.year && item.year !== filters.year) return false;
    // ... 其他筛选条件
    return true;
  });
};
```
