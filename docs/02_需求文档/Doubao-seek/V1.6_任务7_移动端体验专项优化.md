## 一、任务目标

优化移动端体验，新增大字模式、精简信息密度、加载状态提示、渲染性能优化。

---

## 二、详细需求

### 2.1 大字模式

**功能描述**：
- 系统设置新增"大字模式"开关
- 开启后金额、单价、核心合规数据加粗放大

**放大规则**：
| 元素 | 正常模式 | 大字模式 |
|------|---------|---------|
| 金额 | 16px | 24px |
| 单价 | 14px | 20px |
| 合规提示 | 14px | 18px |
| 按钮文字 | 14px | 18px |

**UI设计**：
```
设置页面:
┌─────────────────────────────────────────┐
│ 显示设置                                 │
├─────────────────────────────────────────┤
│ 大字模式                    [开关]      │
│ 金额、单价、合规数据放大显示             │
└─────────────────────────────────────────┘
```

### 2.2 精简信息密度

**优化内容**：
- 减少页面非核心信息展示
- 折叠次要内容（展开/收起）
- 优化手机端自适应布局

**具体调整**：
- 商品列表：隐藏描述，仅显示名称+价格
- 历史方案：折叠详细商品清单
- 合规测算：简化进度条说明文字

### 2.3 加载状态提示

**适用场景**：
- 生成方案时
- 导出文件时
- 加载历史方案时
- 切换页面时

**加载组件**：
```
┌─────────────────────────────────────────┐
│                                         │
│     [Spinner动画]                       │
│                                         │
│     正在生成采购方案...                  │
│     预计需要 3-5 秒                     │
│                                         │
└─────────────────────────────────────────┘
```

### 2.4 渲染性能优化

**优化措施**：
- 长列表虚拟滚动
- 图片懒加载
- 组件按需渲染
- 减少重绘重排
