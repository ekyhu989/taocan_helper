## 一、任务目标

重构底部导航栏，移除「更多」折叠菜单，实现高频功能一键直达，新增合规红色紧急预警按钮。

---

## 二、详细需求

### 2.1 移动端导航重构

**当前问题**：
- 底部导航有「更多」折叠菜单，历史方案、商品库等功能需要二次点击
- 高频功能（历史方案）入口较深

**优化方案**：

| 平台 | 导航项 | 说明 |
|------|--------|------|
| **移动端** | 首页 / 方案 / 合规 / 政策文库 / 商品库 | 固定5项，移除「更多」折叠 |
| **桌面端** | 首页 / 方案 / 合规 / 政策文库 / 商品库 / 历史方案 | 6项，历史方案常驻 |

**实现要求**：
- 移动端始终显示5个导航项（图标+文字）
- 桌面端屏幕宽度>1024px时显示6个导航项
- 当前页面高亮样式保持不变
- 点击切换页面动画保留（300ms ease-in-out）

### 2.2 合规红色紧急预警按钮

**功能描述**：
- 仅在存在合规风险时显示红色闪烁按钮
- 无风险时永久静默（不显示）
- 点击直达合规诊断页

**触发条件**（满足任一即显示）：
1. 年度人均预算 > 2000元
2. 单次采购人均 > 500元
3. 832平台占比 < 30%

**UI要求**：
```
- 位置：导航栏右侧（移动端）/ 顶部导航（桌面端）
- 样式：红色背景 + 白色警告图标 + 闪烁动画
- 文字："⚠️ 合规预警"（移动端简化为图标）
- 点击：跳转到合规诊断页
```

**闪烁动画**：
```css
@keyframes warning-flash {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.6; }
}
animation: warning-flash 1s ease-in-out infinite;
```

### 2.3 合规诊断页

**页面路径**：`/diagnosis` 或 `currentView = 'diagnosis'`

**页面内容**：
1. **风险概览卡片**
   - 显示当前所有风险项数量
   - 红色大字号显示

2. **风险详情列表**
   - 每项风险独立卡片展示
   - 包含：风险类型、当前值、限制值、风险等级
   - 示例：
     ```
     风险：年度人均超标
     当前：2,500元/人
     限制：2,000元/人
     等级：🔴 高风险
     ```

3. **核算逻辑说明**
   - 展开/收起形式展示计算公式
   - 例如：人均 = 总预算 ÷ 人数

4. **一键整改方案按钮**
   - 点击自动调整预算/人数至合规范围
   - 显示调整前后的对比
