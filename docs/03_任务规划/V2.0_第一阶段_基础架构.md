### 任务3：状态管理架构设计

**任务编号**: DeepSeek-V2.0-003  
**分配给**: DeepSeek  
**优先级**: P0  
**预计工时**: 2天

#### 需求描述
设计并实现状态管理架构，支持年度数据隔离、跨组件状态共享。

#### 具体工作
1. 选择状态管理方案（Zustand或Context API）
2. 设计Store结构：
   ```typescript
   // stores/schemeStore.ts
   interface SchemeState {
     currentScheme: Scheme | null;
     schemeHistory: Scheme[];
     draftScheme: Scheme | null;
     
     // Actions
     createScheme: (data: SchemeCreateData) => void;
     updateScheme: (id: string, data: Partial<Scheme>) => void;
     saveDraft: (scheme: Scheme) => void;
     loadDraft: () => Scheme | null;
   }

   // stores/complianceStore.ts
   interface ComplianceState {
     annualBudget: Record<number, BudgetProgress>;
     currentYear: number;
     
     // Actions
     setAnnualBudget: (year: number, budget: number) => void;
     updateUsedBudget: (year: number, amount: number) => void;
     switchYear: (year: number) => void;
   }
   ```

3. 实现年度数据隔离逻辑
4. 实现数据持久化（localStorage）

#### 验收标准
- [ ] Store架构设计完成
