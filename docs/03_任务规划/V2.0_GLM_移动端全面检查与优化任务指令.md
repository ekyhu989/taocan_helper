#### 2.3 历史页面只读版 (MobileHistory)

**功能要求**:
```typescript
// 新建文件: src/pages/mobile/MobileHistory.tsx
// 功能: 只读查看历史方案

interface MobileHistoryProps {
  onNavigate: (page: string) => void;
}

// 功能清单:
- 显示历史方案列表（从localStorage读取）
- 列表项显示: 日期、单位名称、总金额
- 点击列表项 → 查看方案详情（只读）
- 详情页显示: 基本信息、商品清单、合规信息
- 返回按钮 → 返回历史列表
- 不支持编辑（只读）
```

**代码示例**:
```tsx
// src/pages/mobile/MobileHistory.tsx
import React, { useState, useEffect } from 'react';
import { loadHistory } from '../../utils/helpers/historyStorage';

const MobileHistory: React.FC<{ onNavigate: (page: string) => void }> = ({ onNavigate }) => {
  const [historyList, setHistoryList] = useState([]);
  const [selectedItem, setSelectedItem] = useState(null);
  
  useEffect(() => {
    const history = loadHistory();
    setHistoryList(history);
  }, []);
  
  if (selectedItem) {
    return (
      <div className="min-h-screen bg-gray-50">
        {/* 详情页 - 只读 */}
        <div className="bg-white p-4 border-b">
          <button onClick={() => setSelectedItem(null)}>← 返回</button>
          <h1 className="text-xl font-bold ml-2">方案详情</h1>
        </div>
        <div className="p-4 space-y-4">
          {/* 只读显示方案信息 */}
          <div className="bg-white rounded-lg p-4">
            <h3 className="font-bold mb-2">{selectedItem.summary.unitName}</h3>
            <p>日期: {formatDate(selectedItem.createdAt)}</p>
            <p>总金额: ¥{selectedItem.summary.totalAmount}</p>
            <p>人数: {selectedItem.summary.headCount}人</p>
          </div>
          {/* 商品清单 - 只读 */}
          <div className="bg-white rounded-lg p-4">
            <h3 className="font-bold mb-2">商品清单</h3>
            {selectedItem.solutionData.productList.items.map((item, idx) => (
              <div key={idx} className="py-2 border-b last:border-0">
                {item.product.name} × {item.quantity}
              </div>
            ))}
          </div>
        </div>
      </div>
    );
  }
  
  return (
    <div className="min-h-screen bg-gray-50">
      <div className="bg-white p-4 border-b">
        <h1 className="text-xl font-bold">历史方案</h1>
      </div>
      <div className="p-4 space-y-3">
        {historyList.length === 0 ? (
          <div className="text-center text-gray-500 py-8">暂无历史方案</div>
        ) : (
          historyList.map((item) => (
            <div 
              key={item.id}
              onClick={() => setSelectedItem(item)}
              className="bg-white rounded-lg p-4 shadow-sm"
            >
              <div className="flex justify-between items-center">
                <div>
                  <h3 className="font-medium">{item.summary.unitName}</h3>
                  <p className="text-sm text-gray-500">
                    {formatDate(item.createdAt)} · {item.summary.headCount}人
                  </p>
                </div>
                <div className="text-right">
                  <p className="font-bold text-blue-600">
                    ¥{item.summary.totalAmount}
                  </p>
                  <span className="text-xs text-gray-400">点击查看</span>
                </div>
              </div>
            </div>
          ))
        )}
      </div>
    </div>
  );
};
```

**验收标准**:
- [ ] 历史列表正确显示（从localStorage读取）
- [ ] 点击列表项进入详情页
- [ ] 详情页只读，无编辑功能

### 任务3: 路由懒加载实现 (P1)

**目标**: 实现路由懒加载，减少首屏加载时间

**实现文件**: `src/AppV2.jsx`

**实现代码**:
```jsx
import React, { Suspense, lazy } from 'react';

// 懒加载移动端组件
const MobileHomePage = lazy(() => import('./pages/mobile/MobileHomePage'));
const NewScheme = lazy(() => import('./pages/mobile/NewScheme'));
const SolutionList = lazy(() => import('./pages/mobile/SolutionList'));
const MobileHistory = lazy(() => import('./pages/mobile/MobileHistory'));
const MobileBottomNav = lazy(() => import('./components/mobile/MobileBottomNav'));

// 懒加载桌面端组件（可选，移动端优先）
const HomePage = lazy(() => import('./pages/desktop/HomePage'));
const SolutionPage = lazy(() => import('./pages/desktop/SolutionPage'));
// ... 其他桌面端组件

// 加载状态组件
const PageLoading = () => (
  <div className="flex items-center justify-center min-h-screen">
    <div className="animate-spin rounded-full h-12 w-12 border-b-2 border-blue-600"></div>
  </div>
);

// 使用Suspense包裹
<Suspense fallback={<PageLoading />}>
  {deviceType === 'mobile' ? (
    <div className="mobile-container">
      <div className="mobile-content">
        {currentPage === 'home' && <MobileHomePage onPageChange={handlePageChange} />}
        {currentPage === 'scheme' && <NewScheme onNavigate={handlePageChange} />}
        {currentPage === 'solution' && <SolutionList onNavigate={handlePageChange} />}
        {currentPage === 'history' && <MobileHistory onNavigate={handlePageChange} />}
      </div>
      <MobileBottomNav currentPage={currentPage} onNavigate={handlePageChange} />
    </div>
  ) : (
    // 桌面端...
  )}
</Suspense>
```

**验收标准**:
- [ ] 使用 React.lazy 实现组件懒加载
- [ ] 使用 Suspense 包裹，有加载状态
- [ ] 首屏加载时间减少（构建包体积分析）
- [ ] 路由切换流畅
