## 🚀 快速开始

### 启动开发

```bash
# 1. 查看当前阶段任务
cat docs/03_任务规划/V2.0_第一阶段_基础架构.md

# 2. 根据分配开始开发
# DeepSeek: 架构搭建
# GLM: 等待第一阶段完成
# Doubao: 等待第一阶段完成

# 3. 提交任务日志
echo "任务完成" > tasks/{Agent}/V2.0-XXX_任务名称_完成.md
```

### 任务提交流程

```
开发完成
    ↓
创建任务日志（按模板）
    ↓
代码提交（git commit -m "type: V2.0-XXX 描述"）
    ↓
通知项目经理验收
    ↓
通过/打回修改
```
