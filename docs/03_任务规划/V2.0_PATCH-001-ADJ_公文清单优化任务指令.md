## 一、需求描述

### 1.1 当前问题

**当前显示**（红框部分）：
```
三、拟采购物资清单

1. 牛奶礼盒（1箱 × 55元）
2. 优质大米（1袋 × 45元）
3. 蜂蜜礼盒（1盒 × 59.9元）
4. 核桃仁（1袋 × 55元）
5. 大米（1份 × 35元）
```

**问题**：
- 公文清单显示单价（55元、45元等）不合适
- 缺少总体数量说明

### 1.2 优化方案

**改为显示**：
```
三、拟采购物资清单

本次采购优先通过832平台采购脱贫地区农副产品，主要包含食品饮料及相关生活物资，具体如下：

1. 牛奶礼盒（每人1箱，总计50箱）
2. 优质大米（每人1袋，总计50袋）
3. 蜂蜜礼盒（每人1盒，总计50盒）
4. 核桃仁（每人1袋，总计50袋）
5. 大米（每人1份，总计50份）

以上物资每人一份，共计50人份。
```

## 三、代码示例

### 完整的清单渲染代码

```tsx
// 拟采购物资清单部分
<section className="mt-6">

  </p>

      const totalQuantity = item.quantity * personCount;

        <li key={index} className="text-gray-800">
          {item.name}（每人{item.quantity}{item.unit}，总计{totalQuantity}{item.unit}）
        </li>
      );
    })}
  </ol>
  
  {/* 新增汇总说明 */}
  <p className="mt-4 text-gray-700">
    以上物资每人一份，共计{personCount}人份。
  </p>
</section>
```
