智能采购助手 V1.5 功能汇总
一、产品定位
智能采购助手 - 新疆工会采购合规方案生成工具

核心价值：帮助新疆地区机关单位工会工作人员快速生成合规的采购申请报告和物资品单，确保符合工会经费使用政策和832平台采购要求。

二、功能模块总览
1. 首页（Landing Page）- V1.5新增
4个功能入口卡片：
📝 制定方案 → 进入采购流程
📊 合规测算 → 检查年度预算合规性
📚 政策文库 → 查看政策文件
📦 商品库 → 管理采购商品
项目简介：一句话说明产品价值
快速开始：醒目的"立即制定方案"按钮
响应式设计：移动端优先，适配桌面端
2. 采购流程（核心功能）
Page1 - 基础信息录入

采购场景选择（传统节日慰问/专项活动物资/精准帮扶慰问）
采购人数输入
总预算输入
资金来源选择（行政福利费/工会经费/专项慰问费/其他经费）
预算模式（按人均标准/按总额控制）
意向品类（食品/日用品/文体用品/其他节日礼品）
采购地区（默认新疆地区）
节日类型（春节/古尔邦节/肉孜节等，根据场景联动显示）
Page2 - 采购方案预览

智能生成商品清单（4-6个商品随机组合）
自动分配数量和金额
显示合计金额、人均标准
832平台占比提示（绿色提示区域）
合规提醒（人均>500元黄色预警）
消费帮扶提示（建议优先选用832平台产品）
手动调整功能：修改数量、删除商品、添加商品、恢复原始方案
预算使用率进度条
Page3 - 公文生成

单位名称、申请部门、申请人输入
年份、节日选择
一键生成采购申请报告
支持Word导出（.docx格式）
支持PDF导出（.pdf格式）
支持打印（优化打印样式）
示例公文预览
3. 合规测算（Page4）
年度总预算输入
已完成采购金额输入
自动计算完成率（百分比显示）
三色进度条（<30%红色/30-100%黄色/≥100%绿色）
2000元年度上限校验（人均>2000元橙色警告）
还需采购金额计算
政策文库（3个核心政策文档链接）
4. 商品库管理
商品列表展示（名称、单价、单位、分类、832标识）
新增商品
编辑商品
删除商品
搜索和筛选（按名称、按分类）
恢复默认商品库
数据持久化（localStorage存储）
5. 历史方案
自动保存每次生成的采购方案
历史列表（倒序排列，最多50条）
查看历史方案详情
使用历史方案（一键加载全部状态）
删除单条历史
清空所有历史
表单数据持久化（自动保存/恢复）
6. 底部导航栏（V1.5新增）
固定在页面底部
4个导航项：
🏠 首页
📝 方案（快速进入Page1）
📊 合规（快速进入Page4）
➕ 更多（展开菜单：政策文库/商品库/历史方案/设置）
当前页面高亮显示
菜单展开/收起动画
7. 页面切换动画（V1.5新增）
首页→功能页：从右向左滑入（300ms, ease-in-out）
功能页→首页：从左向右滑出（300ms, ease-in-out）
功能页之间：淡入淡出（200ms）
三、技术特点
特性	说明
技术栈	React + Tailwind CSS + Vite + TypeScript
数据存储	localStorage（本地存储，无后端）
响应式	移动端优先（375px），适配桌面端（1440px）
部署	腾讯云云开发（国内CDN加速）
构建体积	~1.5MB（JS+CSS）
四、合规特性
合规要求	实现方式
术语规范	使用"采购/方案/预算/物资"，禁用"购买/下单/支付/商城"
单次预警	人均>500元黄色提示
年度上限	人均>2000元橙色警告
832平台提示	慰问品中832产品占比≥30%绿色提示
政策引用	公文引用"新财购〔2025〕2号"等最新政策文号
