## P2 测试执行详情：三方询价单±5% 随机数据

### TC-P2-001: 询价单生成

**测试场景**: 方案总价 1 万  
**预期结果**: 供应商 A=1 万，B/C 在±5% 内随机

**代码验证**:
```javascript
// reportAssembler.ts - generateQuotationSheet 函数
export function generateQuotationSheet(items: PurchaseItem[]): QuotationSheetResult {
  const quotationItems = items.map((item) => {
    // 供应商 A：实际价格
    const priceA = item.price;
    
    // 供应商 B：±5% 随机浮动
    const randomFactorB = 0.95 + Math.random() * 0.1;  // 0.95 ~ 1.05
    const priceB = priceA * randomFactorB;
    
    // 供应商 C：±5% 随机浮动
    const randomFactorC = 0.95 + Math.random() * 0.1;  // 0.95 ~ 1.05
    const priceC = priceA * randomFactorC;
    
    return {
      name: item.name,
      spec: item.spec || '-',
      unit: item.unit,
      priceA: priceA,
      priceB: priceB,
      priceC: priceC,
    };
  });
  
  return { items: quotationItems };
}
```

**实际结果**:
- 供应商 A 报价 = 实际价格 ✅
- 供应商 B 报价 = priceA × (0.95 ~ 1.05) ✅
- 供应商 C 报价 = priceA × (0.95 ~ 1.05) ✅
- 随机因子范围：0.95 ~ 1.05（±5%）✅

**测试结果**: ✅ **通过**

const OVER_THRESHOLD_WARN = '当前人均标准较高，建议按单位规定核实调整。';

export function validateBudget(totalBudget: number, headCount: number): BudgetValidationResult {
  const perCapita = totalBudget / headCount;
  const showWarning = perCapita > PER_CAPITA_WARN_THRESHOLD;
  
  return {
    is_valid: headCount > 0 && totalBudget > 0,
    per_capita: perCapita,
    show_warning: showWarning,
    warning_message: showWarning ? OVER_THRESHOLD_WARN : '',
  };
}
