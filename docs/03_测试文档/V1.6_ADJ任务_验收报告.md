**代码检查** (`productListGenerator.ts`):
```typescript
function calculateMaxPerCapita(perCapitaBudget: number, fundSource: string): number {
  if (fundSource === '工会经费') {
    return Math.floor(perCapitaBudget / 0.8);
  } else {
    return Math.floor(perCapitaBudget / 0.9);
  }
}
```
